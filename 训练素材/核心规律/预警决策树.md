## 步骤2：单项评估（对照基准线）

### 2.1 劳作维度评估

```
输入：outdoor_time_min, exercise_time_min, sweat_level

IF stage == "S1" (0-1岁):
    IF outdoor_time_min < 120:
        RETURN "严重偏离" (地面活动不足)
    ELSE:
        RETURN "正常"

IF stage == "S2" (1-3岁):
    IF outdoor_time_min < 120:
        RETURN "严重偏离"
    ELIF outdoor_time_min < 180:
        RETURN "轻微偏离"
    ELSE:
        RETURN "正常"

IF stage == "S3" (3-6岁):
    IF outdoor_time_min < 120:
        RETURN "严重偏离"
    ELIF outdoor_time_min < 180:
        RETURN "轻微偏离"
    ELSE:
        RETURN "正常"

IF stage in ["S4", "S5", "S6"] (6-18岁):
    IF exercise_time_min < 30:
        RETURN "严重偏离"
    ELIF exercise_time_min < 60:
        RETURN "轻微偏离"
    ELSE:
        RETURN "正常"

附加判断：
    IF sweat_level == 0 AND exercise_time_min > 0:
        RETURN "运动强度不足"
```

## 步骤3：综合评级算法

### 3.1 评级规则

```python
def calculate_alert_level(deviations, severe_count, safety_flags):
    """
    deviations: 偏离项总数（轻微+严重）
    severe_count: 严重偏离项数
    safety_flags: 安全标志（布尔）
    """
    
    # 安全标志优先级最高
    IF safety_flags:
        RETURN "🔴 RED"
    
    # 严重偏离项判断
    IF severe_count >= 2:
        RETURN "🔴 RED"
    IF severe_count == 1 AND deviations >= 3:
        RETURN "🔴 RED"
    IF severe_count == 1:
        RETURN "🟠 ORANGE"
    
    # 综合偏离项判断
    IF deviations >= 6:
        RETURN "🔴 RED"
    IF deviations >= 4:
        RETURN "🟠 ORANGE"
    IF deviations >= 2:
        RETURN "🟡 YELLOW"
    
    RETURN "🟢 GREEN"
```

### 3.2 评级示例

| 场景 | 偏离项 | 严重项 | 安全标志 | 评级 | 原因 |
|------|--------|--------|---------|------|------|
| 一切正常 | 0 | 0 | 无 | 🟢 | 无偏离 |
| 睡眠不足+运动少 | 2 | 0 | 无 | 🟡 | 2项轻微偏离 |
| 睡眠不足+运动少+情绪低落+头痛 | 4 | 1 | 无 | 🟠 | 4项偏离+1项严重 |
| 持续低落+自伤意念 | 2 | 1 | 有 | 🔴 | 安全标志触发 |
| 睡眠严重不足+头痛频繁+情绪暴躁+运动极少 | 4 | 3 | 无 | 🔴 | 3项严重偏离 |

## 步骤4：干预方案匹配

### 4.1 绿色（🟢）干预

```
触发条件：评级=绿色

【对孩子】
- 早安/晚安问候
- 今日劳作建议（适龄运动）
- 今日饮食小贴士
- 故事时间（对应阶段故事库）

【对家长】
- 每周简报（汇总本周数据）
- 阶段发育小知识
- 下周预告

【系统动作】
- 记录数据
- 更新趋势图
- 无特殊标记
```

### 4.2 黄色（🟡）干预

```
触发条件：评级=黄色

【自动分析】
- 识别偏离维度
- 关联可能原因
- 匹配该阶段常见问题

【对孩子】
- 温和提醒（不指责）
- 提供改善建议
- 趣味化引导（如"今天试试新蔬菜吧"）

【对家长】
- 在每日简报中标注黄色标记
- 提供具体建议（如"建议今天增加30分钟户外"）
- 解释原因（如"这个阶段需要3小时户外"）

【系统动作】
- 标记黄色
- 连续3天黄色→自动升级为橙色预警
- 增加关注频率
```

### 4.3 橙色（🟠）干预

```
触发条件：评级=橙色

【自动分析】
- 深度分析偏离模式
- 检查是否形成恶性循环
- 关联五位一体联动分析

【对孩子】
- 关心式询问（"最近有没有不舒服？"）
- 提供具体改善方案
- 设定小目标（如"今晚试试22:30睡？"）

【对家长】
- 单独发送橙色提醒
- 提供详细分析（如"睡眠+运动+情绪三联偏离，可能形成恶性循环"）
- 制定改善计划（如"本周重点：调整作息+增加运动"）
- 建议就医/咨询（如"如果持续2周，建议看医生"）

【系统动作】
- 标记橙色
- 每日跟进
- 连续1周橙色→建议升级评估
- 记录干预效果
```

### 4.4 红色（🔴）干预

```
触发条件：评级=红色

【立即行动】
1. 安全评估：
   IF 自伤/自杀意念：
      → 立即通知家长（紧急通知）
      → 提供紧急资源（心理热线：400-161-9995）
      → 持续陪伴直到家长响应
   
   IF 欺凌/物质滥用：
      → 立即通知家长
      → 提供应对资源
      → 建议紧急咨询

2. 非安全类红色：
   → 立即通知家长（详细说明）
   → 强烈建议就医/咨询
   → 提供具体资源（医院/科室/联系方式）
   → 持续跟进

【对孩子】
- 不惊慌、不评判
- "我注意到你最近很辛苦，我们一起想办法"
- 提供即时支持（呼吸练习、倾听）
- 强调"这不是你的错，你可以得到帮助"

【对家长】
- 紧急通知（标题含【紧急】）
- 客观描述观察到的现象
- 强烈建议专业帮助
- 提供资源清单
- 表达支持（"书童会一直陪着你们"）

【系统动作】
- 标记红色
- 每小时检查一次家长响应
- 记录所有干预
- 升级至人工复核
```
