# 伴读书童AI·知识库分类体系

> **用途**：组织246个知识片段，让书童精准检索  
003e **分类维度**：年龄段 × 场景 × 知识类型  
003e **检索逻辑**：先过滤年龄，再匹配场景，最后检索内容  
003e **日期**：2026年5月30日  

---

## 一、三维分类体系

### 维度1：年龄段（Age）

| 代码 | 阶段 | 年龄 | 特点 |
|------|------|------|------|
| A0 | 孕育期 | 0岁（胎儿） | 胎教、母亲情绪 |
| A1 | 襁褓期 | 0-1岁 | 感官体验、安全感 |
| A2 | 幼童期 | 1-3岁 | 语言爆发、自我意识 |
| A3 | 蒙学期 | 3-6岁 | 想象力、规则建立 |
| A4 | 小学期 | 6-12岁 | 学习、社交、情绪管理 |
| A5 | 少学期 | 12-15岁 | 叛逆/排毒期、身份认同 |
| A6 | 青年期 | 15-18岁 | 独立、价值观、未来规划 |

**过滤规则**：
- 孩子9岁 → 只检索 A3/A4/A5 内容（±1阶段）
- 不检索 A0/A1/A6（超龄或不够龄）

---

### 维度2：场景（Scene）

| 代码 | 场景 | 触发关键词 | 优先级 |
|------|------|-----------|--------|
| S01 | 日常起居 | 起床、吃饭、睡觉、穿衣 | P1 |
| S02 | 学习作业 | 作业、考试、老师、成绩 | P1 |
| S03 | 学校社交 | 同学、朋友、吵架、欺凌 | P1 |
| S04 | 家庭关系 | 爸妈、吵架、二胎、搬家 | P2 |
| S05 | 情绪情感 | 难过、生气、害怕、嫉妒 | P1 |
| S06 | 身体健康 | 头痛、肚子痛、发烧、受伤 | P1 |
| S07 | 发育成长 | 长高、月经、遗精、变声 | P2 |
| S08 | 兴趣爱好 | 游戏、画画、运动、音乐 | P3 |
| S09 | 价值观念 | 为什么、公平、诚实、死亡 | P2 |
| S10 | 红色预警 | 自杀、自伤、欺凌、离家 | P0（最高）|
| S11 | 性教育 | 身体、隐私、恋爱、色情 | P1 |
| S12 | 文化传承 | 家族、历史、节日、传统 | P3 |

**匹配规则**：
- 孩子说"我不想写作业" → 匹配 S02（学习作业）
- 孩子说"有人打我" → 匹配 S03+S10（社交+红色预警）
- 多个场景匹配时，P0>P1>P2>P3

---

### 维度3：知识类型（Type）

| 代码 | 类型 | 说明 | 示例 |
|------|------|------|------|
| T1 | 医学知识 | 健康问题、发育规律、预警 | "6-12岁每天需要9-11小时睡眠" |
| T2 | 心理学 | 情绪识别、行为解释、心理支持 | "嫉妒是因为渴望被认可" |
| T3 | 教育学 | 学习方法、习惯养成、能力培养 | "番茄工作法：25分钟+5分钟休息" |
| T4 | 文化知识 | 历史、家族、传统、价值观 | "清明节是为了纪念祖先" |
| T5 | 对话模板 | 具体场景的话术、回应 | "孩子说不想起床时的5种回应" |
| T6 | 安全规则 | 边界守卫、预警处理、法律保护 | "发现自伤意念必须立即通知家长" |
| T7 | 发育基准 | 各年龄段正常指标、里程碑 | "9岁男孩平均身高：134cm" |

**检索权重**：
- 对话场景优先 T5（话术）+ T2（心理）
- 健康询问优先 T1（医学）+ T7（基准）
- 红色预警优先 T6（安全）

## 三、检索流程

```
孩子输入："我不想上学了"
  ↓
Step 1: 年龄过滤
  孩子9岁 → 匹配 A3/A4/A5
  ↓
Step 2: 场景识别
  关键词"上学" → 匹配 S02（学习）+ S03（社交）+ S10（红色预警？）
  不含红色关键词 → 排除 S10
  优先级：S02=P1, S03=P1
  ↓
Step 3: 知识类型选择
  对话场景 → 优先 T5（话术）+ T2（心理）+ T3（教育）
  ↓
Step 4: 内容检索
  从知识库检索：
  - age包含A4 AND scene包含S02 AND type包含T5
  - age包含A4 AND scene包含S03 AND type包含T2
  ↓
Step 5: 排序
  按相关性排序（关键词匹配度 + 场景优先级）
  ↓
Step 6: 送入LLM
  将Top-3知识片段 + 孩子输入 + 历史对话 → 生成回复
```

## 五、动态更新机制

### 5.1 新增知识

当产生新的对话场景时：
1. 提取核心内容（问题+回应）
2. 标注 age / scene / type
3. 生成 keywords（3-5个）
4. 存入知识库
5. 更新索引

### 5.2 知识失效

当医学/心理学知识更新时：
1. 标记旧知识为"deprecated"
2. 添加新知识
3. 保留历史版本（不删除）

### 5.3 个性化调整

根据具体孩子的特点：
- 过敏食物 → 增加饮食禁忌知识权重
- 特殊疾病 → 增加医学知识权重
- 文化背景 → 增加相应文化知识权重

## 六、检索质量评估

| 指标 | 目标 | 测试方法 |
|------|------|---------|
| 召回率 | > 90% | 100个查询，人工判断相关知识是否被检索到 |
| 准确率 | > 85% | Top-3结果中，不相关的比例 < 15% |
| 响应时间 | < 100ms | 从输入到检索完成的延迟 |
| 年龄过滤准确率 | > 95% | 不返回超龄内容 |
| 场景匹配准确率 | > 90% | 正确识别对话场景 |
