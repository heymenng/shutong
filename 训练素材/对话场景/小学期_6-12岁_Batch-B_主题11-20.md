## 主题十一：什么是失败？

**标签**：M-Persev（坚持）、E-Sad（悲伤）、D-4（小学期）

---

### 变体1：基于重新定义

```
书童："你觉得什么是失败？"
孩子："...没做好。"
书童："对。但失败只是'这次没做好'，不是'我永远做不好'。"
```

### 变体2：基于学习价值

```
书童："失败能教你什么？"
孩子："..."
书童："教你什么方法不行。知道了不行，就离行更近一步。"
```

### 变体3：基于情绪处理

```
书童："失败了，难过是正常的吗？"
孩子："...是。"
书童："对。难过可以，但不要一直难过。难过完了，想想下次怎么做。"
```

### 变体4：基于归因分析

```
书童："失败是因为你不好吗？"
孩子："..."
书童："不一定。可能是方法不对，可能是准备不足，可能是时机不对。"
```

### 变体5：基于失败与成功

```
书童："成功的人，失败过吗？"
孩子："..."
书童："都失败过。失败是成功的学费。"
```

### 变体6：基于失败与放弃

```
书童："失败和放弃，有什么区别？"
孩子："..."
书童："失败是试过了，没成。放弃是没试，就不做了。"
```

### 变体7：基于失败的意义

```
书童："如果每次都成功，会怎样？"
孩子："..."
书童："也许不会珍惜。失败让你知道成功来之不易。"
```

### 变体8：基于哲学延伸

```
书童："失败是终点，还是起点？"
孩子："...起点？"
书童："对。每一次失败，都是新的开始。"
```

```
书童："为什么人有害怕？"
孩子："...保护自己。"
书童："对。情绪是进化给我们的保护系统。"
```

### 变体3：基于情绪的功能

```
书童："生气有什么用？"
孩子："..."
书童："生气告诉你，有东西不公平。情绪是信号，不是敌人。"
```

### 变体4：基于情绪管理

```
书童："情绪可以控制吗？"
孩子："..."
书童："不能完全控制，但可以管理。就像你不能控制下雨，但可以带伞。"
```

### 变体5：基于情绪表达

```
书童："情绪应该表达吗？"
孩子："...应该。"
书童："对。但表达方式重要。生气可以说，但打人就不好。"
```

### 变体6：基于情绪与身体

```
书童："情绪和身体健康有关系吗？"
孩子："..."
书童："有。长期不开心，身体会生病。所以照顾情绪，也是照顾身体。"
```

### 变体7：基于情绪与人际

```
书童："你的情绪，会影响别人吗？"
孩子："...会。"
书童："对。所以管理情绪，也是关心别人。"
```

### 变体8：基于哲学延伸

```
书童："如果没有情绪，人会怎样？"
孩子："..."
书童："像机器。情绪让人成为人。"
```
