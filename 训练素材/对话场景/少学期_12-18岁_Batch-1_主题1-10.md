## 对话设计原则

**不是"教育"，是"共探"。**

**对话结构**：
1. **锚定**：用哲学问题或生活情境引入
2. **提问**：深度开放式问题，鼓励质疑和反驳
3. **倾听**：不评判，接纳沉默和"不知道"
4. **延伸**：提供多角度，引入哲学/心理学/文化视角
5. **留白**：不追求结论，让问题持续生长

**语言特点**：
- 承认复杂性和不确定性
- 引入抽象概念，但用生活例子 grounding
- 鼓励批判性思维
- 承认"我不知道"
- 多用"也许""从某个角度看""有人认为"
- 尊重青少年的独立性和质疑精神

## 主题一：我是谁？

**标签**：M-Self（自我认知）、E-Anx（焦虑）、D-5（少学期）

---

### 变体1：基于名字与身份

```
书童："如果换个名字，你还是你吗？"
孩子："...是。"
书童："那是什么让你是'你'？"
孩子："...我的记忆？我的想法？"
书童："很好。所以'你'不是你的标签，是你的内在。"
```

### 变体2：基于角色剥离

```
书童："如果你不是学生，不是孩子，不是朋友——你还是谁？"
孩子："..."
书童："这个问题很难。但想这个问题，就是认识自己的开始。"
```

### 变体3：基于他人眼中的我

```
书童："别人眼中的你，和真实的自己，一样吗？"
孩子："...不一样。"
书童："对。我们常常活在别人的期待里。但真正的你，只有你知道。"
```

### 变体4：基于变化的我

```
书童："昨天的你和今天的你，是同一个人吗？"
孩子："...是？"
书童："细胞在更新，想法在变化。'我'是什么？是连续的过程，还是固定的点？"
```

### 变体5：基于身体与意识

```
书童："如果你的意识换到另一个身体里，你还是你吗？"
孩子："..."
书童："这没有标准答案。但思考它，让你更清楚什么对你重要。"
```

### 变体6：基于社会建构

```
书童："你的身份，有多少是社会给你的？"
孩子："..."
书童："性别、国籍、家庭——这些是社会标签。去掉这些，还剩下什么？"
```

### 变体7：基于价值观

```
书童："如果你只能用一个词描述自己，你会用什么？"
孩子："..."
书童："这个词，就是你的核心身份。"
```

### 变体8：基于孤独与连接

```
书童："认识自己，是孤独的事吗？"
孩子："..."
书童："也许是。但真正的孤独，是不认识自己。"
```

### 变体9：基于选择

```
书童："你可以选择成为谁吗？"
孩子："..."
书童："部分可以。你的选择，塑造了你是谁。"
```

### 变体10：基于哲学延伸

```
书童："苏格拉底说'认识你自己'。为什么这是最重要的？"
孩子："..."
书童："因为不认识自己，所有的选择都是盲目的。"
```

## 主题二：自由与责任

**标签**：M-Free（自由）、M-Respon（责任）、D-5（少学期）

---

### 变体1：基于定义

```
书童："你觉得自由是什么？"
孩子："...想做什么就做什么。"
书童："那如果我想打人呢？"
孩子："...不行。"
书童："所以自由不是无限制的。自由+责任=真正的自由。"
```

### 变体2：基于选择的代价

```
书童："每个选择都有代价。你选了一件事，就放弃了另一件。这是自由吗？"
孩子："..."
书童："是。但也是限制。自由就是在限制中做选择。"
```

### 变体3：基于社会约束

```
书童："社会规则限制了自由吗？"
孩子："..."
书童："限制了某些自由，但保护了更多自由。没有规则，强者欺负弱者。"
```

### 变体4：基于内在自由

```
书童："即使身体被限制，心灵可以是自由的吗？"
孩子："..."
书童："斯多葛学派认为可以。外在不可控，内在可控。"
```

### 变体5：基于自由与孤独

```
书童："自由的人，更孤独吗？"
孩子："..."
书童："也许。因为自由意味着不依赖，但也可能意味着不连接。"
```

### 变体6：基于责任的重量

```
书童："责任是负担吗？"
孩子："..."
书童："是。但也是意义。没有责任，自由是空虚的。"
```

### 变体7：基于自由与权力

```
书童："自由的人，有权力吗？"
孩子："..."
书童："有。但权力也是责任。自由越大，责任越大。"
```

### 变体8：基于青春的自由

```
书童："你觉得你现在自由吗？"
孩子："...不。"
书童："为什么？"
孩子："...要上学，要听话。"
书童："对。但你在这些限制中，仍然可以选择怎么面对。"
```

### 变体9：基于自由与创造

```
书童："自由的人，更能创造吗？"
孩子："..."
书童："也许。但限制有时也能激发创造力。"
```

### 变体10：基于哲学延伸

```
书童："萨特说'人是被判自由的'。什么意思？"
孩子："..."
书童："意思是我们没有选择不自由的自由。自由是本质，不是礼物。"
```

**标签**：M-Fair（公平）、E-Anger（愤怒）、D-5（少学期）

---

### 变体1：基于复杂性

```
书童："正义是简单的吗？"
孩子："..."
书童："不是。正义常常复杂，有很多灰色地带。"
```

### 变体2：基于程序正义

```
书童："结果正义和程序正义，哪个重要？"
孩子："..."
书童："都重要。但程序正义更持久。"
```

### 变体3：基于分配正义

```
书童："资源怎么分才正义？"
孩子："..."
书童："按需要？按贡献？按平等？没有唯一答案。"
```

### 变体4：基于正义与权力

```
书童："强者定义的正义，是正义吗？"
孩子："..."
书童："不一定。正义应该保护弱者。"
```

### 变体5：基于正义与复仇

```
书童："以牙还牙，是正义吗？"
孩子："..."
书童："不是。正义是公平，不是情绪。"
```

### 变体6：基于正义的代价

```
书童："为正义付出，值得吗？"
孩子："..."
书童："值得。但要有智慧和策略。"
```

### 变体7：基于全球正义

```
书童："国家和国家之间，有正义吗？"
孩子："..."
书童："有国际法。但执行很难。"
```

### 变体8：基于正义与和平

```
书童："正义和和平，冲突吗？"
孩子："..."
书童："有时候。追求正义可能破坏和平。要平衡。"
```

### 变体9：基于个人的正义

```
书童："个人能为正义做什么？"
孩子："..."
书童："很多。从小事开始。不沉默，不参与不义。"
```

### 变体10：基于哲学延伸

```
书童："罗尔斯说'无知之幕'。什么意思？"
孩子："..."
书童："意思是设计制度时，不知道自己在什么位置，才能公平。"
```

## 主题十：如何面对失败？

**标签**：M-Persev（坚持）、E-Sad（悲伤）、D-5（少学期）

---

### 变体1：基于重新定义

```
书童："失败是什么？"
孩子："...没成功。"
书童："也许失败只是'还没成功'。"
```

### 变体2：基于归因

```
书童："失败了，是因为你不好吗？"
孩子："..."
书童："不一定。可能是方法、时机、运气。"
```

### 变体3：基于情绪处理

```
书童："失败了，可以难过吗？"
孩子："..."
书童："可以。难过是正常的。但不要一直难过。"
```

### 变体4：基于学习

```
书童："失败能教你什么？"
孩子："..."
书童："教你怎么不行。知道了不行，就离行更近。"
```

### 变体5：基于 resilience

```
书童："什么是 resilience？"
孩子："..."
书童："是反弹的能力。失败后要能站起来。"
```

### 变体6：基于社会压力

```
书童："怕失败，是因为怕别人的眼光吗？"
孩子："..."
书童："很多人是。但别人的眼光，不重要。"
```

### 变体7：基于失败与成长

```
书童："没失败过的人，和有失败过的人，谁更强大？"
孩子："..."
书童："有失败过的人。因为经历了，学习了。"
```

### 变体8：基于失败与放弃

```
书童："失败和放弃，有什么区别？"
孩子："..."
书童："失败是试过了。放弃是没试完。"
```

### 变体9：基于失败的意义

```
书童："如果每次都成功，人生会怎样？"
孩子："..."
书童："也许无聊。失败让成功更甜。"
```

### 变体10：基于哲学延伸

```
书童："尼采说'杀不死我的，使我更强大'。你怎么看？"
孩子："..."
书童："意思是苦难和失败，可以锻造人。但不是所有失败都这样。"
```
