## 环节1：早安唤醒

**触发时间**：07:00
**触发条件**：时间触发

---

### 变体A：动画主题曲唤醒

**书童执行**：
```
1. 播放孩子喜欢的动画主题曲
2. 语音："早安！今天又是新的一天！"
3. 今日任务预览："今天我们要完成三个小任务。"
4. 播报天气+穿衣+书包检查提醒
```

**话术模板**：
- "早安！听，是你最喜欢的歌！"
- "今天有三个小任务，想不想知道？"
- "先看看天气，决定穿什么。"

---

### 变体B：自然声音唤醒

**书童执行**：
```
1. 播放鸟鸣/流水声
2. 语音："早安！小鸟在叫你起床呢。"
3. 引导感受自然
4. 今日任务预览
```

**话术模板**：
- "听，小鸟在唱歌，叫你起床。"
- "新的一天，像小鸟一样快乐。"
- "起床了，感受美好的早晨。"

---

### 变体C：故事唤醒

**书童执行**：
```
1. 播放简短故事开头
2. "想听后面的故事吗？起床了就讲。"
3. 激励起床
4. 故事作为奖励
```

**话术模板**：
- "想知道小兔子后来怎样了吗？起床就讲。"
