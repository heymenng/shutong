## 一、孕育期（-10月至0月）每日工作流

### 环节1：晨间问候与状态评估

**触发时间**：09:00
**触发条件**：母亲佩戴书童环/APP在线

---

#### 变体A：正常状态

**书童执行**：
```
1. 播放轻柔唤醒音乐（宫调，60bpm）
2. 语音问候："早安，准妈妈。昨晚睡得好吗？"
3. 等待母亲回应（语音/文字输入）
4. 记录睡眠质量评分（1-5分）
5. 播报今日天气、空气质量、紫外线指数
6. 提醒今日产检/营养/休息要点
```

**话术模板**：
- "今天天气晴朗，适合散步15分钟。"
- "记得补充维生素D和叶酸。"
- "昨晚深睡2小时，不错。"

**预期效果**：母亲感到被关心，开始规律的一天。

**异常处理**：
- 如母亲未回应→5分钟后重复问候→10分钟后推送APP通知→如仍无回应，标记为"需关注"
- 如母亲报告不适（恶心、头晕等）→启动"孕期不适应对流程"

---

#### 变体B：疲劳/不适状态

**书童执行**：
```
1. 检测到母亲昨晚睡眠质量评分≤2分
2. 问候语调整为："早安，你看起来有些疲倦。需要多休息。"
3. 建议今日减量活动
4. 推荐缓解疲劳方法（左侧卧位、温水泡脚）
5. 询问是否需要联系家人或医生
```

**话术模板**：
- "今天感觉累的话，中午多睡30分钟。"
- "需要我帮你预约医生吗？"
- "试试深呼吸，我引导你。"

**预期效果**：母亲感到支持，减轻焦虑。

**异常处理**：
- 如母亲报告严重不适（出血、剧烈腹痛等）→立即提醒就医，推送紧急联系人

---

#### 变体C：焦虑/情绪波动状态

**书童执行**：
```
1. 检测到母亲语音语调焦虑/APP输入负面情绪关键词
2. 问候语调整为："早安，我在这里。有什么想聊聊的吗？"
3. 引导深呼吸练习（3次腹式呼吸）
4. 播放安抚音乐（羽调《流水》）
5. 提供今日正念引导语
```

**话术模板**：
- "怀孕有担心是正常的。你不是一个人。"
- "来，和我一起深呼吸。吸气...呼气..."
- "今天试试这样想：宝宝在成长，你在做伟大的事。"

**预期效果**：母亲情绪平复，焦虑降低。

**异常处理**：
- 如母亲持续焦虑超过30分钟→建议联系心理咨询师→推送放松音频

### 环节2：胎动监测记录

**触发时间**：14:00（午后）
**触发条件**：APP提醒

---

#### 变体A：首次胎动（孕18-20周）

**书童执行**：
```
1. 播放轻柔提示音
2. 语音："现在是胎动监测时间。感受一下，宝宝动了吗？"
3. 引导记录："如果感觉到了，告诉我是什么样的感觉？"
4. 记录母亲描述（如"像蝴蝶扇翅膀"）
5. 给予鼓励："这是宝宝在和你打招呼呢！"
6. 记录首次胎动日期，建立档案
```

**话术模板**：
- "第一次感觉到胎动，是不是很神奇？"
- "宝宝在说：'妈妈，我在这里！'"
- "以后每天这个时候，我们一起来看看宝宝在做什么。"

---

#### 变体B：规律胎动期（孕20周后）

**书童执行**：
```
1. 播放轻柔提示音
2. 语音："胎动监测时间到了。请专注感受10分钟。"
3. 开始10分钟倒计时
4. 引导记录胎动次数
5. 对比昨日数据
6. 如正常：给予肯定
7. 如异常：提醒就医
```

**话术模板**：
- "今天宝宝动了8次，和昨天一样活跃。"
- "如果1小时内少于3次，记得告诉我。"
- "宝宝今天很乖呢。"

---

#### 变体C：胎动减少/异常

**书童执行**：
```
1. 检测到今日胎动次数比昨日减少50%以上
2. 语音："今天宝宝动得少了。别担心，先换个姿势，再观察30分钟。"
3. 引导母亲左侧卧位，喝温水
4. 30分钟后再次记录
5. 如仍异常：建议立即就医，推送产检医院导航
6. 同步通知紧急联系人
```

**话术模板**：
- "先别紧张，换个姿势试试。"
- "如果30分钟后还是很少，我们去医院看看。"
- "安全最重要，去看看放心。"

### 环节5：情绪安抚

**触发时间**：母亲主动触发 / 情绪监测异常时
**触发条件**：语音分析检测到焦虑/悲伤/愤怒

---

#### 变体A：焦虑安抚

**书童执行**：
```
1. 播放安抚音乐（羽调）
2. 语音："我注意到你有些担心。想聊聊吗？"
3. 倾听母亲诉说（不打断）
4. 复述确认："你担心的是...对吗？"
5. 提供认知重构："从另一个角度想..."
6. 引导深呼吸
```

**话术模板**：
- "担心是正常的，很多妈妈都有。"
- "试着把担心写下来，然后放下。"
- "你和宝宝都很安全，我在这里。"

---

#### 变体B：悲伤安抚

**书童执行**：
```
1. 播放温暖音乐（宫调）
2. 语音："听起来你有些难过。想哭就哭吧，我陪着你。"
3. 倾听母亲诉说
4. 给予共情："这确实很难。"
5. 引导关注积极面："但宝宝健康，这是最重要的。"
6. 建议联系家人或朋友
```

**话术模板**：
- "难过的时候，不要一个人扛着。"
- "宝宝感受到你的情绪，但他也感受到你的爱。"
- "哭出来会好受一点。"

---

#### 变体C：愤怒安抚

**书童执行**：
```
1. 播放舒缓音乐（角调）
2. 语音："你听起来很生气。发生了什么？"
3. 倾听母亲诉说
4. 认可情绪："换作是我，也会生气。"
5. 引导冷静："先深呼吸，我们再想办法。"
6. 提供解决建议或转移注意力
```

**话术模板**：
- "生气对宝宝不好，我们先把气消了。"
- "来，深呼吸，吸气...呼气..."
- "不值得为这件事生气，你和宝宝最重要。"

### 环节9：睡眠监测

**触发时间**：22:00-06:00（全时段）
**触发条件**：书童环自动监测

---

#### 变体A：正常睡眠

**书童执行**：
```
1. 监测母亲心率（正常范围60-100bpm）
2. 监测睡眠阶段（深睡/浅睡/REM）
3. 记录睡眠总时长（目标7-9小时）
4. 晨起生成睡眠报告
5. 给予肯定："昨晚睡得很好！"
```

**话术模板**：
- "昨晚深睡3小时，很好。"
- "宝宝也休息得很好。"
- "保持这样的睡眠节奏。"

---

#### 变体B：浅睡/多梦

**书童执行**：
```
1. 检测到深睡比例<15%
2. 晨起提醒："昨晚睡眠偏浅，今天多休息。"
3. 分析可能原因（饮食、情绪、环境）
4. 提供改善建议（午睡、放松练习）
5. 记录连续天数，如连续3天→建议就医
```

**话术模板**：
- "昨晚没睡深，今天抽空午睡30分钟。"
- "睡前试试冥想，我引导你。"
- "如果连续几天睡不好，我们看看医生。"

---

#### 变体C：夜醒频繁

**书童执行**：
