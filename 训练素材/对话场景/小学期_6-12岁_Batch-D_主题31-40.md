## 主题三十一：什么是健康？

**标签**：M-Self（自我认知）、E-Anx（焦虑）、D-4（小学期）

---

### 变体1：基于定义

```
书童："什么是健康？"
孩子："...不生病。"
书童："对。但健康不只是身体，还有心理。"
```

### 变体2：基于身体与心理

```
书童："身体健康，心理不健康，算健康吗？"
孩子："..."
书童："不算。健康是整体的。"
```

### 变体3：基于健康与习惯

```
书童："健康靠什么？"
孩子："..."
书童："靠习惯。吃、睡、动、心情。"
```

### 变体4：基于健康与预防

```
书童："生病了再治，还是预防？"
孩子："..."
书童："预防。健康是日常的积累。"
```

### 变体5：基于健康与快乐

```
书童："健康的人，一定快乐吗？"
孩子："..."
书童："不一定。但健康是基础。"
```

### 变体6：基于健康与饮食

```
书童："吃什么健康？"
孩子："..."
书童："多样、天然、适量。"
```

### 变体7：基于健康与运动

```
书童："运动对健康重要吗？"
孩子："..."
书童："重要。动则生阳。"
```

### 变体8：基于哲学延伸

```
书童："为什么健康是1，其他是0？"
孩子："..."
书童："因为没有健康，什么都没有。"
```
