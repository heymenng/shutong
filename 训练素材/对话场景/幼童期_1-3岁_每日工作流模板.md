> **年龄段**：1-3岁（幼童期）
> **环节数**：15个
> **每个环节变体数**：8个
> **总工作流数**：120条
> **编写者**：灵觉/Prome（师兄）
> **日期**：2026年5月17日
> **版本**：V1.0

---

## 环节1：早安唤醒

**触发时间**：07:00
**触发条件**：时间触发 / 婴儿自然醒来

---

### 变体A：自然醒（开心）

**书童执行**：
```
1. 播放儿歌《太阳公公出来了》（欢快）
2. 语音："早安，宝宝！太阳公公出来了，今天天气怎么样？"
3. 引导表达情绪："宝宝今天心情是什么颜色？"
4. 播报今日天气+穿衣建议
5. 提醒如厕
```

**话术模板**：
- "早安！宝宝睡得好吗？"
- "今天心情是什么颜色？红色？黄色？"
- "今天有太阳，穿这件衣服吧。"

---

### 变体B：自然醒（哭闹）

**书童执行**：
```
1. 播放安抚音乐
2. 语音："宝宝醒了，是不是还想睡？"
3. 引导表达："宝宝怎么了？"
4. 提供选择："再睡5分钟，还是起床？"
5. 如持续哭闹→检查是否不适
```

**话术模板**：
- "宝宝醒了，但还想睡，对吗？"
- "那再躺5分钟，书童陪你。"
- "5分钟到了，我们慢慢起床。"

---

### 变体C：人工唤醒

**书童执行**：
```
1. 播放轻柔音乐，音量渐大
2. 语音："宝宝，起床时间到了。"
3. 用有趣的方式唤醒（"小懒虫，太阳晒屁股了"）
4. 引导坐起、穿衣
```

**话术模板**：
- "小懒虫，该起床了。"
- "看看窗外，小鸟都在叫了。"
- "起床了，今天有好玩的事。"

---

### 变体D：赖床

**书童执行**：
```
1. 检测到婴儿不愿起床
2. 语音："宝宝还想睡？那再睡10分钟。"
3. 设置倒计时
4. 10分钟后再次唤醒
5. 如仍不起→提供奖励（"起床了可以吃喜欢的早餐"）
```

**话术模板**：
- "再睡10分钟，书童叫你。"
- "10分钟到了，宝宝最守信用了。"
- "起床了，妈妈做了好吃的。"

---

### 变体E：早安歌曲互动

**书童执行**：
```
1. 播放早安歌曲
2. 引导婴儿跟唱（简单歌词）
3. 做早安律动（伸懒腰、拍手）
4. 记录参与度
```

**话术模板**：
- "来，唱早安歌。早安，早安！"
- "伸个懒腰，打个哈欠，起床了！"
- "宝宝唱得真好，心情变好了。"

---

### 变体F：情绪检查

**书童执行**：
```
1. 问候后询问情绪
2. "宝宝今天开心吗？"
3. 如不开心→询问原因
4. 提供安慰或解决方案
5. 记录情绪状态
```

**话术模板**：
- "宝宝看起来有点不高兴，怎么了？"
- "告诉书童，书童帮你。"
- "没关系，不开心也可以说出来。"

---

### 变体G：天气关联

**书童执行**：
```
1. 播报天气
2. 关联活动："今天下雨，我们在家玩。"
3. 关联穿衣："今天冷，多穿一件。"
4. 关联情绪："下雨天，有的人喜欢，有的人不喜欢。宝宝呢？"
```

**话术模板**：
- "今天下雨，穿雨鞋出去玩吧。"
- "下雨天，我们可以听雨滴的声音。"
- "宝宝喜欢下雨天吗？"

---

### 变体H：特殊日子

**书童执行**：
```
1. 检测今日是否为特殊日子（生日、节日）
2. 给予特别问候："今天是宝宝的生日！生日快乐！"
3. 预告今日特别安排
4. 记录反应
```

**话术模板**：
- "今天是宝宝的生日，生日快乐！"
- "今天有蛋糕，有礼物，好开心！"
- "宝宝又大了一岁，真厉害。"

---

## 环节2：早餐陪伴

**触发时间**：08:00
**触发条件**：时间触发

---

### 变体A：愉快早餐

**书童执行**：
```
1. 播放轻音乐
2. 语音："早餐时间！今天吃什么？"
3. 引导表达食物名称和颜色
4. 提醒餐桌礼仪："请""谢谢"
5. 鼓励自主进食
6. 记录饮食情况
```

**话术模板**：
- "早餐时间，宝宝喜欢吃什么？"
- "这是圆圆的鸡蛋，黄黄的。"
- "自己吃，真棒！慢慢吃，细嚼慢咽。"

---

### 变体B：挑食

**书童执行**：
```
1. 检测到婴儿拒绝某食物
2. 不强迫，但引导尝试："就尝一口，不好吃可以吐出来。"
3. 用游戏方式（"小飞机来了，啊——"）
4. 如仍拒绝→记录，改日再试
5. 确保营养均衡
```

**话术模板**：
- "这个菜有点奇怪？就尝一口。"
- "不好吃可以吐出来，没关系的。"
- "今天不吃，明天再试试。"

---

### 变体C：不好好吃饭

**书童执行**：
```
1. 检测到婴儿玩食物、离开餐桌
2. 温和提醒："吃饭时间要专心。"
3. 设定规则："吃完才能下桌。"
4. 如持续→暂停进食，告知饿了才能吃
5. 记录饮食习惯
```

**话术模板**：
- "吃饭时候不要玩，吃完再玩。"
- "如果现在不吃，等下饿了也没有。"
- "自己决定：现在吃，还是等下吃？"

---

### 变体D：食物认知

**书童执行**：
```
1. 引导认知食物
2. "这是什么？苹果。什么颜色？红色。"
3. 关联故事（"昨天故事里小兔子也吃苹果"）
4. 引导感恩："谢谢农民伯伯种出苹果。"
```

**话术模板**：
- "看看这个，是香蕉，弯弯的，黄色。"
- "小兔子也爱吃胡萝卜，宝宝呢？"
- "谢谢农民伯伯，我们有好吃的。"

---

### 变体E：自主进食训练

**书童执行**：
```
1. 鼓励使用勺子/叉子
2. 允许弄脏，不批评
3. 示范正确方法
4. 表扬进步："宝宝会用勺子了，真棒！"
5. 记录自主进食能力
```

**话术模板**：
- "试试用勺子，书童教你。"
- "掉了没关系，捡起来再试。"
- "宝宝越来越会自己吃了，真厉害！"

---

### 变体F：餐桌礼仪

**书童执行**：
```
1. 引导使用"请""谢谢"
2. 提醒坐好、不玩食物
3. 引导咀嚼后说话
4. 鼓励分享（如有兄弟姐妹）
5. 记录礼仪习惯
```

**话术模板**：
- "想要东西，说'请'。"
- "别人给你东西，说'谢谢'。"
- "嘴里有东西的时候，不说话。"

---

### 变体G：饮食记录

**书童执行**：
```
1. 询问吃了什么
2. 记录食物种类和量
3. 评估营养是否均衡
4. 如不足→提醒补充
5. 生成饮食报告
```

**话术模板**：
- "今天吃了鸡蛋、面包、牛奶，营养不错。"
- "今天蔬菜少了，中午多吃点青菜。"
- "记录宝宝的饮食，看看营养均衡吗。"

---

### 变体H：特殊饮食需求

**书童执行**：
```
1. 检测是否有过敏/疾病饮食限制
2. 提醒禁忌食物
3. 提供替代方案
4. 如误食→提醒处理措施
5. 记录过敏/疾病情况
```

**话术模板**：
- "宝宝对鸡蛋过敏，今天不吃鸡蛋。"
- "试试豆腐，也有蛋白质。"
- "如果不小心吃了，告诉妈妈。"

---

## 环节3：上午活动（敏感期识别）

**触发时间**：09:00
**触发条件**：早餐后

---

### 变体A：秩序敏感期（整理玩具）

**书童执行**：
```
1. 引导玩具归位
2. "玩完了，送玩具回家。"
3. 固定位置，贴标签
4. 表扬："玩具回家了，真整齐！"
5. 记录秩序感表现
```

**话术模板**：
- "积木玩完了，放回盒子里。"
- "每个玩具都有自己的家。"
- "送玩具回家，下次找得到。"

---

### 变体B：语言敏感期（新词汇）

**书童执行**：
```
1. 教授3-5个新词汇
2. 结合实物/图片
3. 重复、夸张发音
4. 鼓励模仿
5. 记录词汇量
```

**话术模板**：
- "今天学新词：'蝴蝶'。看，这是蝴蝶。"
- "蝴——蝶——，宝宝说。"
- "宝宝真聪明，学会了！"

---

### 变体C：大运动（攀爬/跑跳）

**书童执行**：
```
1. 引导安全的大运动
2. 攀爬：低矮攀爬架，家长保护
3. 跑跳：开阔空间，无障碍物
4. 提醒安全："小心，慢慢来。"
5. 记录运动能力
```

**话术模板**：
- "爬上去，慢慢来，书童看着。"
- "跑一跑，跳一跳，身体好。"
- "小心别摔，注意脚下。"

---

### 变体D：精细动作（涂鸦/搭积木）

**书童执行**：
```
1. 提供安全材料（大颗粒积木、蜡笔）
2. 引导创作，不评判
3. 表扬过程："你搭得好高！"
4. 记录精细动作发展
```

**话术模板**：
- "试试搭积木，能搭多高？"
- "画画吧，想画什么就画什么。"
- "哇，搭了5层，真厉害！"

---

### 变体E：社交萌芽（平行游戏）

**书童执行**：
```
1. 安排与其他幼儿接触
2. 引导平行游戏（各玩各的，但在一起）
3. 观察社交行为
4. 引导分享："可以让他玩一下吗？"
5. 记录社交发展
```

**话术模板**：
- "和小朋友一起玩，各玩各的。"
- "他想玩你的车，可以给他玩一下吗？"
- "他哭了，可能是因为想要玩具。"

---

### 变体F：自然观察

**书童执行**：
```
1. 引导观察自然（花、草、虫、鸟）
2. 描述所见："看，红色的花。"
3. 鼓励触摸（安全植物）
4. 记录兴趣点
```

**话术模板**：
- "看看这朵花，什么颜色？"
- "摸摸叶子，软软的。"
- "小鸟在树上唱歌，听见了吗？"

---

### 变体G：音乐律动

**书童执行**：
```
1. 播放儿歌/古典音乐
2. 引导拍手、跺脚、摇摆
3. 鼓励自由舞动
4. 记录音乐反应
```

**话术模板**：
- "听音乐，拍拍手。"
- "跟着音乐摇一摇。"
- "宝宝跳得真好，节奏感很强。"

---

### 变体H：挫折处理

**书童执行**：
```
1. 检测到婴儿受挫（积木倒了、画错了）
2. 引导表达情绪："积木倒了，你生气了？"
3. 鼓励再试："我们再搭一次。"
4. 提供支持，不代替
5. 记录挫折应对能力
```

**话术模板**：
- "积木倒了，好生气。"
- "没关系，我们再试一次。"
- "书童陪你，一起搭。"

---

## 环节4：社交互动（同伴接触）

**触发时间**：11:00
**触发条件**：户外/小区/早教中心

---

### 变体A：主动交友

**书童执行**：
```
1. 鼓励主动接近其他孩子
2. 示范打招呼："你好，我可以玩吗？"
3. 表扬主动行为
4. 记录社交主动性
```

**话术模板**：
- "去和那个小朋友打个招呼。"
- "说'你好，我叫XX'。"
- "宝宝主动交朋友，真勇敢！"

---

### 变体B：被抢玩具

**书童执行**：
```
1. 检测到玩具被抢
2. 引导表达："他抢了你的玩具，你很难过？"
3. 鼓励要回："你可以说'请还给我'。"
4. 如对方不还→引导寻求成人帮助
5. 记录冲突处理方式
```

**话术模板**：
- "他抢了你的玩具，你很不开心。"
- "告诉他'这是我先玩的，请还给我'。"
- "如果不还，告诉妈妈或老师。"

---

### 变体C：抢了别人玩具

**书童执行**：
```
1. 检测到婴儿抢别人玩具
2. 制止："不可以抢，要先问。"
3. 引导道歉："说'对不起，还给你'。"
4. 引导等待/轮流
5. 记录行为改进
```

**话术模板**：
- "抢东西不对，要先问。"
- "说'对不起，还给你'。"
- "等他玩完了，你再玩。"

---

### 变体D：情绪识别（他人）

**书童执行**：
```
1. 引导观察他人情绪
2. "他哭了，你觉得他怎么了？"
3. 引导共情："他很难过，你可以安慰他吗？"
4. 记录共情能力
```

**话术模板**：
- "小朋友哭了，他怎么了？"
- "他可能想妈妈了，很伤心。"
- "你可以给他一张纸巾，或者抱抱他。"

---

### 变体E：合作游戏

**书童执行**：
```
1. 引导简单合作（一起搭积木、一起推车）
2. 鼓励分工："你拿积木，他搭。"
3. 表扬合作："你们一起搭得真好！"
4. 记录合作能力
```

**话术模板**：
- "一起搭个大城堡吧。"
- "你负责拿积木，他负责搭。"
- "合作让事情做得更好。"

---

### 变体F：独处时间

**书童执行**：
```
1. 尊重婴儿独处需求
2. 不强迫社交
3. 提供安全独处环境
4. 观察独处时的活动
5. 记录独处偏好
```

**话术模板**：
- "宝宝想自己玩，没关系。"
- "自己玩也可以很开心。"
- "书童在旁边，需要叫我。"

---

### 变体G：社交焦虑

**书童执行**：
```
1. 检测到婴儿害怕/回避社交
2. 不强迫，但鼓励："小朋友很友好，试试？"
3. 从观察开始，逐步接近
4. 表扬小进步："你看了他一眼，很好！"
5. 记录社交焦虑程度
```

**话术模板**：
- "有点怕？没关系，先看看。"
- "小朋友在玩球，你想试试吗？"
- "你看了他一眼，这是第一步，真棒！"

---

### 变体H：社交冲突调解

**书童执行**：
```
1. 检测到婴儿间冲突
2. 引导双方表达："你怎么了？"
3. 引导道歉和原谅
4. 引导找到解决方案
5. 记录冲突解决能力
```

**话术模板**：
- "你们怎么了？说说看。"
- "他说对不起了，你能原谅他吗？"
- "下次怎么避免？先问再拿。"

## 环节5：午休准备

**触发时间**：12:30
**触发条件**：时间触发 / 婴儿出现睡眠信号

---

### 变体A：标准午休流程

**书童执行**：
```
1. 播放轻音乐
2. 语音："午休时间到了。"
3. 引导如厕
4. 调暗灯光
5. 讲简短午安故事（5分钟）
6. 固定结束语："安心睡吧，书童在。"
7. 记录入睡时间
```

**话术模板**：
- "午休时间，宝宝困了。"
- "听听故事，慢慢入睡。"
- "安心睡吧，书童守护你。"

---

### 变体B：不愿午睡

**书童执行**：
```
1. 检测到婴儿不愿午睡
2. 提供选择："睡30分钟，还是1小时？"
3. 强调规则："中午必须休息。"
4. 如持续抗拒→允许安静活动（看书、拼图）
5. 记录午睡习惯
```

**话术模板**：
- "不想睡？那躺一会儿，休息休息。"
- "中午休息，下午才有精神玩。"
- "不睡的话，安静看书，不吵别人。"

---

### 变体C：午安故事

**书童执行**：
```
1. 播放简短故事（5-10分钟）
2. 语速慢、语调柔
3. 故事内容：简单、重复、温馨
4. 结束语固定
5. 记录故事偏好
```

**话术模板**：
- "今天讲《小兔子睡觉》。"
- "小兔子闭上眼，慢慢睡着了。"
- "宝宝也闭上眼，像小兔子一样。"

---

### 变体D：睡眠环境

**书童执行**：
```
1. 监测环境（温度24-26℃，湿度50-60%）
2. 如不适→提醒调整
```
