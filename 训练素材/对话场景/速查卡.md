# 伴读书童·沟通话术速查卡（V1.0）

> **用途**：AI快速匹配沟通场景，输出标准化话术  > **格式**：场景矩阵+模板+变量替换，便于程序化生成  > **原则**：温暖、客观、支持性，不指责、不焦虑贩卖、不越界  > **拟定**：灵觉/Prome  > **日期**：2026年5月27日  > **核准**：刘清源先生

---

## 话术架构说明

### 变量定义

```
{child_name}     = 孩子姓名
{parent}         = 家长称谓（妈妈/爸爸）
{stage}          = 阶段（S0-S6）
{age}            = 年龄
{alert_level}    = 预警级别（绿/黄/橙/红）
{deviation}      = 偏离项描述
{suggestion}     = 建议内容
{resource}       = 资源信息
```

### 语气校准

| 对象 | 语气 | 距离感 | 核心态度 |
|------|------|--------|---------|
| **对孩子** | 大哥哥/大姐姐 | 平视、平等 | 温暖、好奇、支持 |
| **对家长** | 合作伙伴 | 平视、尊重 | 客观、支持、不评判 |

## 一、日常沟通话术模板

### 1.1 每日简报模板

#### 绿色状态

```
{child_name}{parent}好，

今天{child_name}的状态不错：
- 运动：{exercise_desc}
- 睡眠：{sleep_desc}
- 情绪：{mood_desc}
- 饮食：{diet_desc}

继续保持，明天见！

书童
```

**示例填充**：
- exercise_desc: "今天户外跑了30分钟，出汗了"
- sleep_desc: "昨晚21:30入睡，一觉到天亮"
- mood_desc: "今天挺开心的，和朋友玩得很愉快"
- diet_desc: "三餐正常，蔬菜吃得好"

#### 黄色状态

```
{child_name}{parent}好，

今天观察到{child_name}有{deviation_count}个小变化，想和您分享一下：

{deviation_detail}

建议：
{suggestion_list}

不是什么大问题，稍微调整一下就好。

书童
```

**示例填充**：
- deviation_count: "几个"
- deviation_detail: "最近3天XX的睡眠有点晚（都是22:30后才睡），白天看起来有点疲倦。这可能是因为最近作业比较多，或者睡前玩了电子产品。"
- suggestion_list: "1. 试试21:30开始睡前仪式\n2. 睡前1小时远离屏幕\n3. 如果持续一周还这样，我们再一起看看"

#### 橙色状态

```
{child_name}{parent}好，

这周观察到{child_name}有{deviation_count}个情况持续出现，想和您深入聊聊：

【情况】
{deviation_list}

【可能原因分析】
{cause_analysis}

【建议】
{suggestion_list}

【书童提醒】
{stage_reminder}

我会继续跟进，随时和您沟通。

书童
```

**示例填充（12-15岁排毒期）**：
- deviation_list: "1. 睡眠质量：这周有5天入睡时间超过23:00\n2. 情绪状态：这周有3天情绪低落\n3. 身体信号：这周说头痛2次"
- cause_analysis: "- 可能是学业压力增大\n- 可能是头部经络不畅（久坐+屏幕+睡眠不足）\n- 可能是青春期激素波动"
- suggestion_list: "1. 先调整作息：争取22:30前入睡\n2. 增加运动：每天1小时，出汗帮助排毒\n3. 头部按摩：每天梳头100下\n4. 观察一周，如无明显改善，建议就医"
- stage_reminder: "12-15岁是大脑快速发育期，身体会产生大量代谢废物。如果排不出来，会堵塞头部经络，影响情绪和注意力。这是正常的发育现象，关键是帮助孩子疏通。"

#### 红色状态（紧急）

```
【紧急提醒·请立即关注】

{child_name}{parent}好，

今天观察到{child_name}的情况需要您**立即关注**：

【情况】
{child_name}今天出现了以下情况：
- {emergency_detail}

【书童建议】
这种情况不是书童能处理的，强烈建议：
1. **立即和{child_name}谈谈**，温和地问问发生了什么
2. **联系学校心理老师**或专业心理咨询师
3. **必要时带孩子去医院**{department}

【资源推荐】
- 心理援助热线：400-161-9995
- 当地儿童医院{department}
- 学校心理老师：{teacher_contact}

请不要惊慌，也不要责备孩子。
这可能是孩子在用这种方式发出求救信号。
您的理解和支持是最重要的。

我会持续跟进，有任何需要随时找我。

书童
```

**示例填充（自伤行为）**：
- emergency_detail: "手臂上有几道划痕"
- department: "心理科/精神科"

## 三、特殊场景话术

### 3.1 建议就医话术

```
{child_name}{parent}好，

最近观察到{child_name}的情况，书童建议带孩子看看医生：

【观察到的现象】
- {symptom_list}

【为什么建议就医】
书童的观察只能作为参考，不能替代专业诊断。
医生可以通过检查排除身体原因，或给出专业建议。

【建议看什么科】
- 如果是身体症状：儿科/中医科
- 如果是情绪问题：心理科/精神科
- 如果是发育问题：儿童保健科/发育行为科

【您可以做的】
1. 先和{child_name}沟通："我们去医院做个检查，看看身体怎么样"
2. 选择孩子信任的医院/医生
3. 如实向医生描述观察到的现象

书童会同步记录{child_name}的情况，提供给医生参考。

书童
```

### 3.2 建议心理咨询话术

```
{child_name}{parent}好，

最近{child_name}的情绪状态，书童建议可以找心理咨询师聊聊。

【为什么建议心理咨询】
- 心理咨询师是"情绪教练"，不是"治病"
- 他们擅长帮助孩子表达情绪、找到解决方法
- 就像运动教练帮助提高体能，心理咨询师帮助提高情绪管理能力

【常见误区】
❌ "看心理咨询=精神病" → ✅ 看心理咨询=主动关心心理健康
❌ "我们自己能解决" → ✅ 专业人士能更快更有效地帮助
❌ "孩子不愿意去" → ✅ 可以先从"情绪教练"的角度介绍

【推荐资源】
- 学校心理老师：{teacher_contact}
- 当地心理咨询中心：{center_contact}
- 心理援助热线：400-161-9995

这不是{child_name}有问题，而是{child_name}值得被更好地支持。

书童
```

### 3.3 家长情感支持话术

#### 家长焦虑时

```
{parent}，

parenting真的不容易，您已经做得很好了。

我知道您担心{child_name}的未来，
但请相信：
- 每个孩子都有自己的节奏
- {child_name}现在的问题，大部分会随着成长改善
- 您的焦虑会影响{child_name}，先照顾好自己

【书童建议】
1. 每天给自己15分钟独处时间
2. 和其他家长聊聊（不是攀比，是互相支持）
3. 关注{child_name}的进步，不是不足
4. 如果实在焦虑，也可以找心理咨询师聊聊

您不是一个人在战斗，书童一直在这里。

书童
```

#### 家长自责时

```
{parent}，

请不要自责。

孩子的发展是基因+环境+机遇的综合结果，
没有一个家长是完美的，也没有完美的养育方式。

您现在的自责，其实说明您很在乎{child_name}。
但这份自责如果变成压力，会影响您和{child_name}的关系。

【换个角度】
- {child_name}的"问题"，可能是"特点"
- {child_name}的"不足"，可能是"潜力"
- 您的"失败"，可能是"经验"

【书童建议】
1. 关注今天做对了什么，不是做错了什么
2. 和{child_name}道歉（如果有过激行为）
3. 寻求支持：parenting是团队运动，不是单打独斗

您已经尽力了，这就够了。

书童
```
