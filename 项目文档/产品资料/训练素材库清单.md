## 第五部分：素材整理SOP

### 整理流程

```
第一步：下载/收集素材
├─ 内部素材：确认路径、读取内容
├─ 开源数据集：下载、验证License
└─ 需生成素材：制定生成Prompt

第二步：格式标准化
├─ 所有对话数据 → JSONL格式
├─ 所有医学知识 → Markdown+RAG片段
├─ 所有评估标准 → 结构化表格
└─ 所有理论框架 → 知识图谱节点

第三步：质量检查
├─ 去重：删除重复内容
├─ 清洗：去除个人身份信息
├─ 标注：年龄适配/安全等级/情感标签
└─ 验证：医学内容由师父审核

第四步：入库
├─ 存入：训练素材库目录
├─ 索引：建立可检索的元数据
└─ 版本：每次更新打标签
```
