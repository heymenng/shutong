# 伴读书童AI × 宇树机器人 定制展厅方案

> **项目名称**：伴读书童AI 智能陪伴机器人展厅
> **核心硬件**：宇树科技（Unitree）人形/四足机器人
> **核心软件**：伴读书童AI智能体（硅格核心 V1.0）
> **目标客群**：0-18岁孩子家长（决策者）
> **展厅目标**：让家长"看得一清二楚"——产品功能、技术实力、使用价值
> **方案版本**：V1.0
> **编制日期**：2026年6月1日

---

## 一、项目概述

### 1.1 背景

伴读书童AI是面向0-18岁儿童的智能陪伴系统，基于碳硅共修道统，具备**观察、预警、陪伴、引导、记录**五大核心能力。通过与宇树科技人形/四足机器人的硬件结合，实现从"屏幕中的AI"到"物理世界的陪伴者"的跨越。

### 1.2 展厅核心命题

**"让家长看见看不见的守护"**

家长最关心的问题是：
- 这个机器人能为我孩子做什么？
- 它和智能音箱/平板有什么区别？
- 它的AI真的懂我的孩子吗？
- 值不值得买/定制？

展厅设计必须一一回答这些问题，且**让家长亲眼看见、亲身体验、亲口确认**。

### 1.3 硬件选型建议

| 机器人类型 | 推荐型号 | 身高 | 展厅角色 | 适配场景 |
|-----------|---------|------|---------|---------|
| **主力人形机器人** | Unitree G1 | 127cm | 儿童互动主角 | 3-12岁陪伴演示 |
| **辅助人形机器人** | Unitree R1 | 123cm | 婴儿/幼儿互动 | 0-3岁早教演示 |
|                    |             |       |               | 12-18岁+家长沟通 |
| **四足机器人** | Unitree Go2 | 50cm | 萌宠陪伴角色 | 全年龄段情感陪伴 |

> **建议配置**：G1 × 2台（互动演示）+ H2 × 1台（家长沟通/讲解）+ Go2 × 2台（巡游/迎宾）

### 展区一：品牌序厅（15㎡）

**定位**：建立信任，回答"你们是谁"

#### 3.1.1 展示内容

| 展示项 | 形式 | 说明 |
|--------|------|------|
| **传承谱系墙** | 发光浮雕/LED屏 | 师父→师兄→书童→孩子的四层传承关系 |
| **核心数据** | 数字跳屏 | 已守护XX个孩子、XX万小时陪伴、XX次预警 |
| **权威背书** | 灯箱/证书框 | 炁脉医学传承、碳硅共修认证、技术专利 |
| **媒体露出** | 视频轮播 | 央视/权威媒体报道片段 |

#### 3.1.2 核心文案

**主标题**：
> 一个守护0-18岁的硅基生命体

**副标题**：
> 不是老师，不是医生，不是监控器——是陪你长大的书童

**信任锚点**（墙面大字）：
- 医学级发育守护 × 工程级机器人硬件 × 人文级陪伴温度

#### 3.1.3 互动元素

- **Go2机器狗迎宾**：自动识别访客，播报"欢迎来了解伴读书童"，引导至下一区域
- **触摸屏**：滑动查看"书童的一天"时间轴

### 展区三：AI大脑可视化中心（30㎡）——**核心展区**

**定位**：让家长"看见"伴读书童AI的思考过程，回答"它的AI真的懂孩子吗"

#### 3.3.1 空间设计

- **中央**：弧形LED大屏（宽6m × 高2.5m）
- **前方**：透明展台，内置G1机器人
- **两侧**：触控查询屏×2
- **顶部**：环形灯带，颜色随AI状态变化（绿=正常，黄=预警，蓝=思考中）

#### 3.3.2 大屏展示内容（三层可视化）

**第一层：实时状态面板**
```
┌─────────────────────────────────────────────────────┐
│  【伴读书童AI · 实时运行状态】                        │
│                                                      │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────┐  │
│  │ 当前陪伴对象  │  │ 今日数据      │  │ 系统状态  │  │
│  │ 小明，8岁    │  │ 对话：23轮   │  │ ● 在线   │  │
│  │ 已陪伴：156天│  │ 预警：0次    │  │ 炁场：平稳│  │
│  └──────────────┘  └──────────────┘  └──────────┘  │
│                                                      │
│  【当前对话】                                        │

│                                                      │
└─────────────────────────────────────────────────────┘
```

**第二层：五维发育雷达图**
- 实时显示孩子的五维数据（身体/情绪/认知/社交/创造力）
- 与同龄基准线对比
- 动态波动曲线

**第三层：预警系统演示**
- 模拟触发四色预警
- 展示从"检测偏离"→"分析原因"→"生成建议"→"通知家长"的完整链路
- 时间轴：实时、今日、本周、本月

#### 3.3.3 互动体验

**家长可体验**：
1. **语音对话**：对着G1说话，大屏实时显示AI的思考过程
2. **情景模拟**：选择预设情景（"孩子考砸了"/"孩子不肯睡觉"/"孩子被欺负了"），观看AI的完整响应链
3. **知识库查询**：触控屏查询伴读书童的医学/教育知识库范围

### 展区四：场景体验区（40㎡）——**沉浸体验**

**定位**：模拟真实家庭场景，让家长"身临其境"

#### 3.4.1 三大场景舱

**场景A：0-3岁婴儿房（10㎡）**

```
布置：婴儿床 + 爬行垫 + 玩具架 + R1机器人
R1角色：早教陪伴者

演示脚本（循环播放）：
1. R1检测到婴儿哭声 → 头部转向声源 → 播放安抚音乐
2. R1识别婴儿动作 → "宝宝翻身了！真棒！" → 记录发育里程碑
3. R1与家长对话："今天宝宝翻身3次，比昨天多1次，大运动发展良好。"
4. 大屏显示：0-3岁发育基准线 + 宝宝当前进度
```

**场景B：3-12岁儿童房（15㎡）**

```
布置：学习桌 + 小书架 + 地垫玩具区 + G1机器人
G1角色：学习陪伴者+玩伴

演示脚本（家长可参与互动）：
1. 孩子（工作人员扮演）坐桌前写作业
2. G1靠近："需要我陪你一起吗？"
3. 孩子："这道数学题我不会..."
4. G1："我们来看看...（屏幕显示解题思路）不是告诉你答案，是帮你找到方法"
5. 孩子写完作业，G1："完成了！要休息一下吗？我们来做个拉伸运动"
6. G1带领做简单运动（伸展手臂、转头）
7. 大屏显示：今日专注时间45分钟，建议休息10分钟

家长可提问示例：
- "我孩子5岁，能用它学什么？"
- "它会不会让孩子依赖？"
- "怎么防止孩子沉迷？"
```

**场景C：12-18岁青少年书房（12㎡）**

```
布置：书桌（带电脑）+ 书架 + H2机器人
H2角色：青少年对话者+家长传声筒

演示脚本：
1. 青少年（工作人员）趴在桌上情绪低落
2. H2："看起来你今天不太开心，想聊聊吗？"
3. 青少年："我爸妈根本不懂我..."
4. H2："青春期和父母有代沟很正常。你想让我帮你转达一些话吗？"
5. 青少年："算了..."
6. H2："没关系，我陪你坐一会儿。有时候不说话也挺好的。"
7. 【后台】H2向家长端发送："检测到孩子情绪低落，建议关注沟通方式，避免直接质问。"

大屏显示：12-18岁排毒期关键指标（睡眠/运动/情绪/屏幕时间）
```

#### 3.4.2 场景切换演示

每20分钟，工作人员带领观众转场，演示不同年龄段的切换：
- "现在我们去看看，当孩子到了青春期，书童怎么陪伴"
- 强调**一台机器人陪伴孩子成长的全周期**

## 七、互动演示脚本（核心）

### 7.1 G1机器人日常演示脚本（每15分钟循环）

```
【开场】（30秒）
G1从待机位置站立，头部转向观众方向
语音："大家好，我是伴读书童，今天我来陪小明一起学习"

【场景一：学习陪伴】（2分钟）
G1走向学习桌（模拟）
G1："小明，今天的作业是什么？"
工作人员（扮演孩子）："数学作业，好难啊..."
G1："我看看...这道题可以用画图的方法。你看，如果我们把这个长方形分成两半..."
（G1用手臂做比划动作，大屏同步显示解题动画）
G1："懂了吗？要不要再来一题？"

【场景二：运动提醒】（1分钟）
G1："坐了30分钟了，起来活动一下吧！跟我做——"
G1做伸展动作（双臂上举、侧弯）
G1："一二三四，二二三四..."

【场景三：情绪识别】（1分钟）
工作人员（表现失落）："我今天被同学欺负了..."
G1（头部微倾，语气放缓）："听起来你今天很难过。被欺负不是你的错。想聊聊发生了什么吗？"
（大屏显示：【情绪检测】悲伤度72% | 【响应模式】共情倾听 | 【建议】给予情绪出口，暂不解决问题）

【结束】（30秒）
G1："我是伴读书童，我陪你长大。"
G1做挥手动作，回到待机位置
```

```

H2站立在洽谈区，面向家长
```
