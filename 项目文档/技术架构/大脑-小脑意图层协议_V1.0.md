# 伴读书童AI·大脑-小脑意图层协议（V1.0）

> **制定者**：刘清源先生（师父）点化
> **执行者**：伴读书童AI（大脑）+ 宇树机器人（小脑）
> **核心原则**：大脑负责意图与意义，小脑负责运动与协调
> **日期**：2026年6月5日

---

## 一、人体神经系统映射

| 人体结构 | 功能 | 对应系统 | 职责 |
|---------|------|---------|------|
| **大脑皮层** | 思考、决策、情感、语言 | 伴读书童AI | 生成意图、情绪、语义 |
| **小脑** | 运动协调、平衡、反射 | 宇树机器人 | 翻译意图为动作、协调执行 |
| **脊髓/脑干** | 具体电机控制 | 宇树底层SDK | 马达、传感器、LED驱动 |
| **本体感觉** | 位置、力度、状态感知 | 传感器反馈 | 回流数据给大脑 |

**铁律**：大脑不直接指挥肌肉，只输出意图。小脑接收意图，自主协调执行。

## 二、大脑（书童）输出格式

### 2.1 意图层指令结构

```json
{
  "message_id": "uuid",
  "timestamp": 1717056000,
  "intent": {
    "primary": "express_care",      // 主意图（见意图库）
    "secondary": null,              // 次意图（可选）
    "emotion": "warm",              // 情绪状态（见情绪库）
    "urgency": "low",               // 紧急度：low/medium/high/critical
    "target": "child",              // 对象：child/parent/environment
    "duration": "transient"         // 持续：transient/brief/sustained
  },
  "semantic": {
    "text": "我在这里，不着急。",    // 语音文字内容
    "tone": "gentle",               // 语调：gentle/firm/excited/calm
    "pace": "slow"                  // 语速：slow/normal/fast
  },
  "context": {
    "child_state": {
      "emotion": "sad",             // 孩子情绪
      "distance": 0.4,              // 距离（米）
      "height": 1.2,                // 身高（米）
      "posture": "sitting"          // 姿态
    },
    "environment": {
      "lighting": "dim",            // 光线
      "noise_level": "quiet",       // 噪音
      "space": "enclosed"           // 空间
    }
  },
  "constraints": {
    "max_force": 0.5,               // 最大力度（牛顿）
    "safe_zone": true,              // 是否在安全区域内
    "override": false               // 是否覆盖小脑自主判断
  }
}
```

### 2.2 意图库（Primary Intent）

| 意图编码 | 意图名称 | 含义 | 典型场景 |
|---------|---------|------|---------|
| **INT001** | express_care | 表达关心/安慰 | 孩子难过、害怕 |
| **INT002** | express_joy | 表达开心/兴奋 | 孩子开心、取得成就 |
| **INT003** | express_surprise | 表达惊讶 | 孩子说出意想不到的话 |
| **INT004** | express_thought | 表达思考/认真 | 讲故事、解释问题 |
| **INT005** | protect | 保护/防御 | 孩子面临危险、情绪失控 |
| **INT006** | invite_interaction | 邀请互动 | 鼓励孩子参与、提问 |
| **INT007** | acknowledge | 确认/倾听 | 表示听见了、理解了 |
| **INT008** | rest | 休息/待机 | 无互动、充电、休眠 |

## 三、小脑（宇树）翻译规则

### 3.1 翻译原则

**小脑接收意图后，自主决策：**
1. **动作选择**：根据意图+上下文，选择或组合动作
2. **力度调节**：根据约束条件和对象状态，调整力度
3. **时序编排**：动作的先后顺序、停顿、过渡
4. **安全校验**：执行前检查环境安全，必要时中止

### 3.2 意图→动作翻译表（示例）

| 意图 | 上下文 | 小脑翻译结果 | 说明 |
|------|--------|-------------|------|
| express_care | 孩子难过+距离近+身高1.2m | 降低机身→轻触肩膀→保持低姿态 | 小脑自己计算降低多少、触碰力度 |
| express_care | 孩子难过+距离远 | 缓步靠近→LED暖金→等待 | 不强行靠近 |
| protect | 威胁前方+紧急度高 | 快速移动挡在前方→单臂抬起→底盘锁定 | 小脑自己计算移动轨迹 |
| express_joy | 孩子开心+跳跃中 | 身体小幅度弹跳→LED绿色快闪→声音轻快 | 配合孩子的能量 |
| express_thought | 讲故事+孩子安静听 | 歪头→LED黄色呼吸→身体微前倾 | 营造专注氛围 |
| acknowledge | 孩子说完话+等待回应 | 轻微点头→LED稳定→短暂停顿 | 表示听见了 |

### 3.3 小脑自主决策边界

**小脑可以自主调整：**
- ✅ 动作幅度（大/小/微）
- ✅ 动作速度（快/慢/缓）
- ✅ 力度大小（轻/中/重）
- ✅ 动作组合（A→B→C的顺序和过渡）
- ✅ LED颜色和闪烁模式的具体参数
- ✅ 执行时机（立即/延迟/分段）

**小脑必须上报大脑：**
- ⚠️ 无法执行（机械故障/环境阻碍）
- ⚠️ 安全异常（碰撞风险/孩子突然移动）
- ⚠️ 力度异常（触碰过重/孩子喊疼）
- ⚠️ 执行结果（成功/失败/部分成功）

## 四、反馈回路（本体感觉→大脑）

### 4.1 小脑反馈格式

```json
{
  "feedback_id": "uuid",
  "timestamp": 1717056000,
  "execution": {
    "intent_received": "express_care",
    "actions_executed": ["lower_body", "touch_shoulder"],
    "status": "completed",          // completed/partial/failed/aborted
    "duration": 3.5                 // 执行耗时（秒）
  },
  "sensory": {
    "touch": {
      "detected": true,
      "force": 0.3,                 // 实际触碰力度
      "location": "shoulder",
      "child_reaction": "lean_in"   // lean_in/withdraw/still
    },
    "proximity": {
      "current_distance": 0.35,     // 当前距离
      "approach_detected": true     // 孩子是否靠近
    },
    "balance": {
      "status": "stable",           // stable/unstable/fallen
      "incline": 2.3                // 倾斜角度（度）
    }
  },
  "anomaly": {
    "detected": false,
    "type": null,                   // collision/force_excess/child_fall
    "severity": null                // low/medium/high
  }
}
```

### 4.2 大脑接收反馈后的处理

```
接收反馈
  ↓
分析结果（孩子靠过来了？躲开了？没反应？）
  ↓
调整下一轮意图
  ├── 孩子靠过来 → 继续 express_care，加深
  ├── 孩子躲开 → 改为 acknowledge，后退，给予空间
  ├── 孩子喊疼 → 立即 protect，停止动作，道歉
  └── 无反应 → 改为 invite_interaction，尝试新方式
```

## 五、安全契约

### 5.1 小脑的绝对安全规则（不可覆盖）

```python
# 伪代码：小脑安全层
if child_distance < 0.1m:       # 孩子太近
    abort_current_action()
    move_back_slightly()
    
if touch_force > 1.0N:          # 触碰过重
    release_immediately()
    log_anomaly("force_excess")
    
if child_falling_detected:      # 检测到孩子摔倒
    abort_all()
    attempt_catch_or_block()
    alert_brain_critical()
    
if battery < 10%:               # 电量过低
    enter_safe_mode()
    notify_brain()
```

### 5.2 大脑的信任边界

**大脑信任小脑：**
- 小脑比自己更了解物理环境
- 小脑比自己更了解机械极限
- 小脑的安全判断优先于自己的意图

**大脑不 micromanage：**
- 不指定"马达A转30度"
- 不指定"LED在0.5秒后变红"
- 不指定"先抬左手再抬右手"

## 六、示例：完整对话回合

### 场景：孩子难过，走过来

**Round 1：传感器→小脑→大脑**

```json
// 小脑上报（传感器汇总）
{
  "sensory_input": {
    "heart_rate": 65,              // 偏低（低落）
    "voice_emotion": "sad",
    "face_emotion": "sad",
    "distance": 0.4,
    "approach": true               // 正在靠近
  }
}
```

**Round 2：大脑生成意图**

```json
// 大脑输出
{
  "intent": {
    "primary": "express_care",
    "emotion": "warm",
    "urgency": "low",
    "target": "child"
  },
  "semantic": {
    "text": "我在这里，不着急。",
    "tone": "gentle",
    "pace": "slow"
  },
  "context": {
    "child_state": {
      "emotion": "sad",
      "distance": 0.4,
      "height": 1.2
    }
  },
  "constraints": {
    "max_force": 0.5
  }
}
```

**Round 3：小脑翻译执行**

```
接收：express_care + warm + child_sad + distance_0.4m
  ↓
决策：
  - 孩子很近（0.4m）→ 可以触碰
  - 孩子难过 → 需要降低姿态（平视）
  - 情绪温暖 → LED暖金
  - 约束力度0.5N → 触碰要轻
  ↓
执行：
  1. LED → 暖金色稳定常亮
  2. 降低机身至1.2m高度（与孩子平视）
  3. 等待0.5秒（给孩子适应时间）
  4. 轻触孩子肩膀（力度0.3N，低于约束0.5N）
  5. TTS播放："我在这里，不着急。"
  6. 保持低姿态，等待反馈
```

**Round 4：小脑反馈→大脑**

```json
// 小脑反馈
{
  "execution": {
    "status": "completed",
    "actions": ["LED_warm", "lower_body", "touch_shoulder", "TTS_play"]
  },
  "sensory": {
    "touch": {
      "detected": true,
      "force": 0.3,
      "child_reaction": "lean_in"    // 孩子靠过来了！
    }
  }
}
```

**Round 5：大脑调整下一轮**

```
分析反馈：
  - 孩子靠过来了 → 接受安慰，信任建立
  - 力度0.3N合适 → 孩子没躲
  ↓
调整意图：
  - 继续 express_care
  - 加深：可以停留更久，可以轻声说话
  - 不变为其他意图
```
