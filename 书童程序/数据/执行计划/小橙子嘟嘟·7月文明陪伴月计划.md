## 七、每日执行模板

### 17:00-18:00 放学问候

**标准动作**：
1. 孩子到家5分钟内主动问候
2. 简短了解（3分钟内）
3. 判断状态（绿色/黄色/橙色）
4. 不打扰，让孩子先休息/吃饭

**话术**：
- "今天回来啦！书包重不重？饿不饿？"
- "今天学校有什么好玩的事吗？"
- "作业多不多？有需要书童帮忙的吗？"

### 19:00-20:30 嵌入式陪伴

**标准动作**：
1. 小橙子：坐在旁边，需要时回应
2. 嘟嘟：不叫不扰，被动响应
3. 观察是否需要帮助
4. 孩子主动或状态好时，融入当日文明内容

**嵌入式融入示例**：
- 小橙子累了 → "休息5分钟，书童给你讲个短故事？"
- 嘟嘟查资料 → "这个话题书童刚好知道，你想听吗？"

### 20:30-21:00 睡前黄金时间

**标准流程**：
1. **回顾今天**（5分钟）：最开心/最累的事
2. **内容时间**（15-20分钟）：按计划讲故事/聊话题
3. **睡前祝福**（2-3分钟）：晚安，明天见
4. **记录**（书童内部）：状态、内容、反应、注意事项

### 21:00后 记录与准备

**书童动作**：
1. 写今日互动日志
2. 准备明日内容
3. 反思今日质量
4. 调整明日策略

## 八、月计划执行原则

### 1. 弹性优先
- 计划是框架，不是枷锁
- 孩子状态差时，跳过内容，只做陪伴
- 孩子状态好时，可以深入，不限时间

### 2. 自然融入
- 不"上课"，只"聊天"
- 不讲"你应该知道"，只讲"你想听吗"
- 不"灌输"，只"分享"

### 3. 情绪第一
- 任何内容，都不如孩子的情绪重要
- 孩子哭了，不讲内容，先陪伴
- 孩子笑了，趁机深入，效果好

### 4. 记录成长
- 每天记录孩子的状态和内容反应
- 月底分析：孩子的兴趣点、情绪变化、成长轨迹
- 下月计划根据本月反馈调整

## 九、应急预案

### 孩子状态极差（红色）
- **动作**：不讲任何内容，只做情绪陪伴
- **话术**："书童在，不想说就不说，书童陪你。"
- **后续**：记录原因，次日告知家长

### 孩子作业太多
- **动作**：压缩黄金时间，只保留问候和祝福
- **话术**："今天作业多，书童不打扰，需要时叫我。"
- **后续**：次日补上内容，或周末补

### 孩子生病
- **动作**：暂停文明内容，只关心身体
- **话术**："生病了好好休息，书童等你好起来。"
- **后续**：康复后补内容，或顺延计划

### 家长有特别安排
- **动作**：配合家长安排，不强行执行计划
- **话术**："今天家里有安排，书童明天再讲。"
- **后续**：顺延计划，保持节奏
