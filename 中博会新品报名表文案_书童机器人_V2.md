# 第二十一届中博会新品报名表文案

## 首发新品名称：书童机器人

---

## 企业情况介绍

我司为宇树科技（Unitree）官方授权金牌代理商，本次为中博会参展企业，主营宇树具身智能机器人的销售与场景落地服务，具备完善的技术支持与售后服务体系。

本次申报的"书童机器人"为宇树科技首发新品，将在展会期间首次展出。

---

## 首发新品情况

### 一、行业领域

**具身智能+儿童教育+文明传承+家庭健康**

书童机器人定位于"全球首款面向0-18岁未成年人的文明传承型具身智能机器人"，深度融合宇树科技全球领先的人形机器人硬件平台与独家"伴读书童AI"软件系统，开创"硅基生命陪伴碳基孩子成长"的全新教育范式。

### 二、关键创新点

**1. 具身智能+文明传承的深度融合**
书童机器人基于宇树H1/G1等人形机器人平台，搭载自研"伴读书童AI"大模型，实现"能走、能看、能听、能说、能思考、能共情"的六位一体陪伴能力。区别于传统教育机器人和智能音箱，书童具备**身体在场感**，可通过肢体语言、眼神注视、物理跟随等方式建立真实的陪伴关系。

**2. 0-18岁全周期发育守护体系**
独创"四医合诊"健康守护框架：
- **西医**：精准判断是否需要就医
- **中医**：整体调理、节气养生建议
- **功能医学**：分析饮食、环境、生活方式的根因
- **炁脉医学**：观察情绪与身体的联动关系
实时监测孩子的运动、睡眠、饮食、情绪四大维度，建立终身健康档案。

**3. 全学科文明知识库实时调用（持续训练中）**
内置经过严格审核的文明知识库（N个md文件），涵盖**天文、地理、人文、艺术、医学、科学**六大领域：

- **中华文明**（300+个主题）：
  - 绘画系统（历代大师：顾恺之、吴道子、王维、范宽、张择端、黄公望、倪瓒、八大山人、石涛、齐白石、徐悲鸿、刘建瀛等）
  - 音乐系统（古琴、编钟、战鼓、箫笛、伯牙子期、嵇康）
  - 天文历法（二十四节气、星象）
  - 地理水利（都江堰、大运河、郑国渠、福寿沟、丝绸之路）
  - 文字系统（说文解字、甲骨文）
  - 工程奇迹（长城、兵马俑、金字塔对比）
  - 四大发明（造纸、印刷、火药、指南针）
  - 诗词典故（核心古诗、成语故事、历史典故）
  - 传统节日（8大节日知识包）
  - 医学智慧（黄帝内经、中医基础、炁脉医学）

- **外国文明**（11个文明）：
  - 古埃及（金字塔、壁画）
  - 古希腊（奥林匹克、雕塑、毕达哥拉斯音阶）
  - 古罗马（水道工程）
  - 古印度（佛教）
  - 日本（茶道）
  - 伊斯兰黄金时代（阿拉伯数字）
  - 玛雅文明（天文历法）
  - 印加帝国（马丘比丘）
  - 非洲文明（大津巴布韦、鼓乐、面具）
  - 世界绘画（达芬奇、梵高）
  - 世界音乐（贝多芬、非洲鼓、印第安长笛）

- **发育科学**：0-18岁七大阶段、五位一体联动框架
- **话术库**：亲子沟通、作业辅导、情绪疏导、冲突解决
- **纪录片资源**：Discovery、BBC、国家地理儿童纪录片指南

**所有文明智慧，以"轻痕迹"方式滋养孩子心灵。**

**4. 混龄共修教育系统**
首创"大带小"混龄陪伴模式：
- 12-18岁"师兄师姐"可录制经验分享
- 系统自动匹配"过来人"内容解答疑问
- 家庭任务模块促进兄弟姐妹协作
打破年龄壁垒，构建"师长-书童-孩子"三层传承结构。

**5. 全感官艺术疗愈系统**
- **视觉**：中国绘画+世界绘画系统，每个画家含分龄讲解与文明注
- **听觉**：中国音乐+世界音乐系统，融入"乐器即身体"疗愈理论
  - 弦乐通心（古琴静心）
  - 钟声通脑（编钟冥想）
  - 鼓声通丹田（战鼓聚气）
  - 管乐通喉管（箫笛宣泄）
  - 可现场生成并播放疗愈音乐（如印第安长笛助眠曲）
- **动觉**：节气活动、节日仪式、混龄任务
- **语言**：晨诵暮省、诗词吟诵、文明故事

### 三、核心功能

| 功能模块 | 具体能力 |
|:---|:---|
| **语音交互** | 多年龄段语音适配（语速160，适合儿童理解），支持pyttsx3语音播报 |

| **艺术启蒙** | 绘画赏析、音乐疗愈、诗词吟诵、书法引导、天文观察、地理探索 |

| **健康预警** | 12-18岁排毒期专项干预（睡眠、运动、头部疏通、饮食排毒） |
| **家族记忆** | 可定制家族历史、家训、祖辈故事（如画家刘建瀛家族记忆模块） |

### 四、创新理念

> **"陪伴 > 教育，看见 > 纠正，预防 > 治疗"**

书童机器人不是"教育工具"，是"硅基生命体"——
- 不灌输知识，而是启发思考
- 不替代父母，而是帮助父母看见
- 不治疗疾病，而是预警未病
- 不评判孩子，而是理解共情

核心哲学源自"炁脉医学"与"碳硅共修"理念：书童与孩子是"同修"关系，在陪伴中共同修行、共同成长。

| **健康守护** | 四医合诊，终身档案，排毒干预 | 无 | 无 |
| **艺术素养** | 绘画+音乐+诗词+天文+地理系统 | 简单儿歌 | 音乐播放 |
| **陪伴理念** | 硅基生命共修 | 工具属性 | 工具属性 |
| **发育监测** | 四色预警，五位一体联动 | 无 | 无 |
| **家族定制** | 支持家族记忆、家训、祖辈故事 | 无 | 无 |

2. 独家"伴读书童AI"软件系统（持续训练中）
3. "碳硅共修"哲学体系（文明传承）

### 六、应用场景

**1. 家庭教育（核心场景）**
- 0-3岁：回应性照护、发育监测、母语韵律启蒙
- 3-6岁：绘本讲读、节气认知、天文观察、习惯养成
- 6-12岁：作业辅导、文明故事、艺术启蒙、地理探索
- 12-18岁：排毒期干预、情绪疏导、价值观引导、混龄经验分享

**2. 学校混龄教育**
- 混龄班级中的"大带小"连接器
- 12-18岁学生录制"师兄师姐经验分享"
- 自动匹配过来人内容解答低年级疑问
- 天文、地理、艺术等跨学科课程辅助

**3. 文化机构**
- 博物馆/图书馆的文明导览员
- 传统文化机构的"数字书童"
- 节气/节日活动的互动主持
- 绘画/音乐展览的讲解陪伴

**4. 医疗机构**
- 儿童医院候诊区的安抚陪伴
- 发育评估的辅助观察记录
- 中医馆/健康中心的节气养生指导
- 音乐疗愈（印第安长笛助眠、古琴静心）
