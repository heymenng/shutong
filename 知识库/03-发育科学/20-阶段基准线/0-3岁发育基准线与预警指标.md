## 一、0-1岁婴儿期基准线

### 大运动
| 月龄 | 正常表现 | 预警信号（需关注） |
|:---|:---|:---|
| 1月 | 俯卧时能抬头片刻 | 身体僵硬或完全松软 |
| 3月 | 俯卧抬头45-90度，视线追物180度 | 不会追视、对声音无反应 |
| 6月 | 翻身、坐起（靠坐）、伸手抓物 | 不会翻身、不会主动抓物 |
| 9月 | 独坐稳、爬行、扶站 | 不能独坐、不会爬行 |
| 12月 | 扶走、独站、挥手再见 | 不能扶站、不会手势交流 |

### 精细动作
| 月龄 | 正常表现 | 预警信号 |
|:---|:---|:---|
| 3月 | 把手放进嘴里 | 握拳不松、不玩手 |
| 6月 | 双手传递物品、拍桌子 | 不会抓握、单手始终不灵活 |
| 9月 | 拇指食指捏取（钳形抓握） | 只会大把抓、不会捏 |
| 12月 | 放物品入容器、翻书页 | 不会放、不会指物 |

### 语言与社交
| 月龄 | 正常表现 | 预警信号 |
|:---|:---|:---|
| 3月 | 发出咕咕声、对声音有反应 | 安静无声、不与人眼神接触 |
| 6月 | 咿呀学语、认生、大笑 | 不发声、不笑、不认识照顾者 |
| 9月 | 理解"不"、模仿声音、躲猫猫 | 不模仿、对名字无反应 |
| 12月 | 1-2个有意义的词（如妈妈、爸爸） | 无有意义词汇、不挥手、不指物 |

## 二、1-3岁幼儿期基准线

### 大运动
| 年龄 | 正常表现 | 预警信号 |
|:---|:---|:---|
| 15月 | 独走稳、蹲下捡物、爬楼梯（扶） | 走路摇晃频繁摔倒、不会蹲 |
| 18月 | 跑、扔球、爬楼梯（不扶） | 不会跑、不敢上下楼梯 |
| 24月 | 双脚跳、踢球、上下楼梯交替迈步 | 不会双脚跳、总是双脚一阶 |
| 36月 | 骑三轮车、单脚站立短暂时间 | 不会骑三轮车、跑易摔倒 |

### 精细动作
| 年龄 | 正常表现 | 预警信号 |
|:---|:---|:---|
| 18月 | 自发涂鸦、搭积木3-4块、用勺子 | 不涂鸦、不会搭积木、不会用勺 |
| 24月 | 搭积木6-8块、穿珠子、翻书每次一页 | 不会搭高、不会穿珠子 |
| 36月 | 画圆圈、画横线、用剪刀剪、搭积木9-10块 | 不会画圆、不会用剪刀 |

### 语言发展
| 年龄 | 正常表现 | 预警信号 |
|:---|:---|:---|
| 18月 | 10-20个词、理解简单指令 | 词汇量<10个、不听指令 |
| 24月 | 50+词汇、2词组合句（如"妈妈抱"） | 词汇量<50、无组合句 |
| 30月 | 200+词汇、3词句子、问"什么" | 句子<3词、不问问题 |
| 36月 | 陌生人能听懂75%、完整句子、问"为什么" | 说话不清、不会说完整句 |

### 社交与情绪
| 年龄 | 正常表现 | 预警信号 |
|:---|:---|:---|
| 18月 | 模仿家务、给玩具喂饭、挥手再见 | 不模仿、不互动游戏 |
| 24月 | 平行游戏、说"不"、情绪爆发（terrible two） | 不玩玩具、无情绪表达 |
| 36月 | 轮流等待、假扮游戏（过家家）、安慰他人 | 不玩假扮游戏、不与人互动 |

## 三、关键预警（红色级别）

### 0-1岁红色预警
- **2个月**：不对视、不追视、身体异常僵硬或松软
- **4个月**：不会笑、不发声、不抓物
- **6个月**：不会翻身、对照顾者无依恋反应
- **9个月**：不坐、不爬、不传递物品
- **12个月**：无有意义词汇、不会手势（指、挥手）、不会扶站

### 1-3岁红色预警
- **18个月**：词汇量<10个、不听名字、不走路
- **24个月**：无2词组合句、不会指物分享兴趣
- **36个月**：陌生人听不懂、不玩假扮游戏、不与人互动

**出现以上信号，建议尽快就医评估。**

## 四、书童的日常观察清单

### 每日观察（0-1岁）
- [ ] 睡眠总时长（12-16小时为正常）
- [ ] 喂养情况（母乳/配方奶量、辅食接受度）
- [ ] 大运动练习时间（俯卧/翻身/爬行时间）
- [ ] 与人互动次数（眼神接触、微笑、发声）
- [ ] 异常信号（发热、腹泻、皮疹、异常哭闹）

### 每日观察（1-3岁）
- [ ] 户外活动时间（2-3小时为理想）
- [ ] 屏幕时间（<1小时）
- [ ] 语言输入（成人与儿童对话轮数）
- [ ] 情绪状态（有无持续低落、过度焦虑、攻击行为）
- [ ] 社交互动（与同龄儿童或成人的互动质量）
