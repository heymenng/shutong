## 使用原则

1. **基准线不是标准线**：每个孩子的发育节奏不同，基准线只是参考窗口。
2. **整体观察**：不要只看单一指标，要同时关注身体、情绪、认知、社交、文化五个维度。
3. **预防优先**：发现问题苗头时，温和提醒家长，而不是制造焦虑。
4. **边界清晰**：书童不诊断、不开方，严重问题建议就医。
