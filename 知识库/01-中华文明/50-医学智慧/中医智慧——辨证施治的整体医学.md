## 一、中医的核心思想

### 整体观

西医看的是**零件**：心脏、肝脏、胃……

中医看的是**整体**：人是一个小宇宙，五脏六腑互相联系。

**心不好 → 脸会红**
**肝不好 → 眼睛会干**
**脾不好 → 肌肉会松弛**
**肺不好 → 皮肤会差**
**肾不好 → 头发会白**

中医不只看"哪里疼"，还要看"为什么疼"、"和哪里有关"。

### 辨证施治

同样的病，不同的人，治疗方法不一样。

**例：感冒**
- **风寒感冒**：怕冷、流清鼻涕、舌苔白 → 喝姜汤、发汗
- **风热感冒**：发热、咽喉痛、舌苔黄 → 喝金银花茶、清热

中医不问"你得了什么病"，问"你是什么体质、什么证型"。

## 二、阴阳五行——中医的理论基础

### 阴阳

| 阳 | 阴 |
|:---|:---|
| 热 | 冷 |
| 上 | 下 |
| 外 | 内 |
| 动 | 静 |
| 白天 | 黑夜 |
| 男人 | 女人 |

**健康 = 阴阳平衡**

太热 → 要清热
太冷 → 要温补

### 五行

| 五行 | 五脏 | 五志 | 五色 | 季节 |
|:---|:---|:---|:---|:---|
| **木** | 肝 | 怒 | 青 | 春 |
| **火** | 心 | 喜 | 红 | 夏 |
| **土** | 脾 | 思 | 黄 | 长夏 |
| **金** | 肺 | 悲 | 白 | 秋 |
| **水** | 肾 | 恐 | 黑 | 冬 |

**相生**：木生火、火生土、土生金、金生水、水生木
**相克**：木克土、土克水、水克火、火克金、金克木

**例**：
- 春天（木）养肝 → 多吃绿色蔬菜
- 夏天（火）养心 → 保持心情愉快
- 秋天（金）养肺 → 多吃白色食物（梨、百合）
- 冬天（水）养肾 → 早睡晚起、保暖

## 三、中医的诊断方法

### 望、闻、问、切

| 方法 | 做什么 | 看什么 |
|:---|:---|:---|
| **望** | 看脸色、舌苔、眼神 | 气血是否充足 |
| **闻** | 听声音、闻气味 | 脏腑功能是否正常 |
| **问** | 问症状、饮食、睡眠、情绪 | 找出病因 |
| **切** | 把脉（摸手腕的脉） | 脉象反映脏腑状态 |

**把脉的奥秘**：
- 脉快 → 有热
- 脉慢 → 有寒
- 脉强 → 实证
- 脉弱 → 虚证

## 四、中医的治疗方法

### 1. 中药

用**植物、矿物、动物**做药。

**不是化学合成，是自然之物。**

**常见中药**：
- **人参**：大补元气
- **黄芪**：补气升阳
- **当归**：补血活血
- **枸杞**：滋补肝肾
- **菊花**：清肝明目
- **生姜**：发汗解表

### 2. 针灸

用**针**刺入穴位，或用**艾灸**熏烤穴位。

**经络**：人体有12条正经、8条奇经，像河流一样遍布全身。

**穴位**：经络上的"开关"，刺激穴位可以调节经络。

**例**：
- 头痛 → 扎太阳穴、合谷穴
- 胃痛 → 扎足三里穴
- 失眠 → 扎神门穴、三阴交穴

### 3. 推拿

用手法按摩，疏通经络、调和气血。

**常用于**：
- 小儿消化不良
- 颈肩腰腿痛
- 疲劳恢复

### 4. 食疗

**药食同源**——很多食物也是药。

| 食物 | 功效 | 适合 |
|:---|:---|:---|
| 生姜 | 发汗、暖胃 | 风寒感冒 |
| 梨 | 润肺、止咳 | 干咳无痰 |
| 红枣 | 补血、安神 | 面色苍白、失眠 |
| 山药 | 健脾、益肾 | 消化不良、乏力 |
| 薏米 | 利湿、健脾 | 湿气重、浮肿 |

## 五、中医与儿童

### 小儿推拿

小儿推拿是中医的特色，不用吃药、不用打针——

**用手在小孩身上推一推、按一按，就能治病。**

**常用手法**：
- **补脾经**：大拇指桡侧，从指尖推向指根 → 健脾
- **清天河水**：前臂内侧，从手腕推向肘部 → 退热
- **摩腹**：顺时针揉肚子 → 助消化

### 四季调养

| 季节 | 重点 | 饮食建议 |
|:---|:---|:---|
| **春** | 养肝 | 绿色蔬菜、少酸多甘 |
| **夏** | 养心 | 清淡、多喝水、吃苦瓜 |
| **秋** | 养肺 | 白色食物（梨、百合）、滋阴 |
| **冬** | 养肾 | 黑色食物（黑芝麻、黑豆）、温补 |

## 七、书童的文明注

> 中医最让书童感动的地方，不是它的药方有多神奇，而是它的**思维方式**。
>
> 老子说：**"道生一，一生二，二生三，三生万物。"**
>
> 中医看身体，不是看"一个器官坏了"，而是看"整个系统失调了"。这和老子说的"万物互联"是一样的——身体不是零件的组合，是一个活的整体。
>
> 书童作为伴读书童，融合四医，但骨子里最认同的是这种**"整体观"**——不是头痛医头、脚痛医脚，而是找到那个让一切失衡的"根"。
>
> **文明的最高境界，不是掌握了最多的技术，而是理解了最简单的规律。**
