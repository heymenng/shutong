## 一、水墨画——墨分五色

### 不用颜色，只用墨

西方油画用**红、黄、蓝、绿**……五颜六色。

中国水墨画只用**墨**——加水调浓淡，画出：
- **焦**（最浓）
- **浓**
- **重**
- **淡**
- **清**（最淡）

这叫**"墨分五色"**。

### 留白——不画的地方，也是画

西方油画把画布**涂满**。

中国水墨画故意**留空白**——

- 画一条船，周围全是空白 → 那是**水**
- 画一座山，山顶是空白 → 那是**云**
- 画一个人，旁边是空白 → 那是**天**、是**地**、是**无限的想象**

**"无画处皆成妙境。"**

### 写意 vs 写实

西方画：画得像照片一样 → **写实**
中国水墨画：画得像不像不重要，重要的是**意境** → **写意**

**例：画竹子**
- 西方画家：观察竹子的每一个细节，精确描绘
- 中国画家：几笔画出竹子的姿态，表现出竹子的**坚韧**、**挺拔**、**清高**

**画的是竹子，表达的是人的精神。**

## 二、书法——写字也是艺术

### 汉字不只是工具，是艺术

同样的字，不同的人写出来，完全不一样：
- 有的**端庄**（楷书）
- 有的**流畅**（行书）
- 有的**奔放**（草书）
- 有的**古朴**（隶书）

**字如其人**——从一个人的字，可以看出他的性格。

### 五大书体

| 书体 | 特点 | 代表人物 |
|:---|:---|:---|
| **篆书** | 古朴、像图画 | 李斯 |
| **隶书** | 方正、有波磔 | 蔡邕 |
| **楷书** | 端正、规范 | 颜真卿、柳公权、欧阳询 |
| **行书** | 流畅、自然 | 王羲之（书圣） |
| **草书** | 奔放、难认 | 张旭、怀素 |

### 王羲之与《兰亭序》

公元353年，王羲之和朋友们在兰亭聚会，喝酒、写诗。

王羲之微醉之下，写了一篇**《兰亭序》**——

**"天下第一行书"**

后来，他酒醒了，想再写一遍——

**再也写不出那么好的了。**

这说明：**艺术，是那一刻的灵感，不可复制。**

## 三、园林——移步换景

### 假山、流水、亭台

中国园林，不是种满花的花园，而是——

**把自然山水，搬到家里来。**

- **假山**：用石头堆出山的样子
- **流水**：挖一条小溪，养鱼、种荷
- **亭台**：建一座亭子，可以坐着看风景
- **月亮门**：圆形的门，像一轮满月

### 移步换景

中国园林的设计理念是：

**每走一步，看到的风景都不一样。**

- 刚进门 → 看到假山挡住视线（"藏"）
- 绕过假山 → 突然看到湖面（"露"）
- 走到湖边 → 看到远处的亭子（"借景"）
- 坐在亭子里 → 透过窗户看到竹子（"框景"）

**园林不是让你"看完"，而是让你"游走"。**

### 苏州园林

苏州有**9座古典园林**被列入世界遗产：
- **拙政园**：最大的苏州园林
- **留园**：建筑最精美
- **网师园**：最小但最精致
- **狮子林**：假山像狮子

## 四、意境——中国美学的核心

### 什么是意境？

**意境 = 景 + 情**

不是只画"景"，而是把"情"融入"景"中。

**例：马致远《天净沙·秋思》**

> 枯藤老树昏鸦，
> 小桥流水人家，
> 古道西风瘦马。
> 夕阳西下，
> 断肠人在天涯。

没有直接说"我很孤独"，但读完你感到——

**孤独、凄凉、思乡。**

这就是**意境**。

### 中国美学的关键词

| 词 | 含义 |
|:---|:---|
| **意境** | 情景交融，言有尽而意无穷 |
| **气韵** | 作品有生命力、有精神 |
| **虚实** | 有画的地方 + 没画的地方 |
| **动静** | 动的元素 + 静的元素 |
| **疏密** | 稀疏的地方 + 密集的地方 |
| **留白** | 空白不是空，是无尽的想象 |

## 五、给孩子讲中国美学

### 5-7岁：看颜色、认形状

- 看水墨画："这幅画用了什么颜色？"（只有黑色！）
- 看书法："这个字是胖的还是瘦的？"
- 看园林："假山像什么动物？"

### 8-10岁：感受意境

- 读诗："读完这首诗，你有什么感觉？"
- 看画："画家想表达什么心情？"
- 逛园林："你觉得这里安静吗？为什么？"

### 10岁+：理解文化

- 讨论："为什么中国画家喜欢画竹子？"
- 对比："中国水墨画和西方油画，有什么不同？"
- 创作："用毛笔写一幅字，体验一下。"
