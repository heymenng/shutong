## 一、二十四节气的由来

### 古人怎么知道时间？

很久很久以前，没有日历、没有钟表，古人怎么知道什么时候播种、什么时候收获？

古人抬头看**太阳**和**星星**。

他们发现：
- 太阳在天空中的位置，每天都稍微移动一点
- 一年下来，太阳走了一个大圆圈
- 把这个圆圈分成24等份，每份大约15天，就是**一个节气**

### 土圭测影

古人用**土圭**（一根直立的杆子）测量日影的长度：
- **夏至**：日影最短，白天最长
- **冬至**：日影最长，白天最短
- **春分/秋分**：日影中等，昼夜平分

通过测量日影，古人确定了**二十四节气**。

## 三、节气与农耕

### 春播
- **立春**：准备农具，翻土
- **雨水**：小麦返青，需要水
- **惊蛰**：春雷响，虫子醒，开始播种
- **春分**：播种黄金期，"春分麦起身"
- **清明**：种瓜点豆
- **谷雨**：雨生百谷，插秧好时节

### 夏长
- **立夏**：小麦抽穗
- **小满**：麦粒渐满，快要成熟
- **芒种**：收小麦，种水稻（"忙种"）
- **夏至**：夏收夏种大忙季节
- **小暑/大暑**：除草、施肥、防虫害

### 秋收
- **立秋**：玉米、高粱成熟
- **处暑**：收获中稻
- **白露**：棉花采摘、花生收获
- **秋分**：秋收秋种，"秋分种麦正当时"
- **寒露**：收割晚稻
- **霜降**：收红薯、萝卜入窖

### 冬藏
- **立冬**：小麦冬灌
- **小雪/大雪**：土地封冻，农闲开始
- **冬至**："冬至大如年"，庆祝丰收
- **小寒/大寒**：积肥、修水利，为来年做准备

## 四、节气与养生

### 春生（养肝）
- **立春-谷雨**：早睡早起，多户外活动
- **饮食**：青色食物（菠菜、芹菜），少吃酸（酸收敛，不利肝发散）
- **情绪**：不生气，肝喜条达

### 夏长（养心）
- **立夏-大暑**：适当出汗，中午小憩
- **饮食**：苦味食物（苦瓜、莲子），红色食物（红枣、番茄）
- **情绪**：保持愉悦，不过度兴奋

### 秋收（养肺）
- **立秋-霜降**：早睡早起，收敛心神
- **饮食**：白色食物（梨、百合、银耳），少吃辛辣
- **情绪**：不悲伤，肺主悲

### 冬藏（养肾）
- **立冬-大寒**：早睡晚起，保存能量
- **饮食**：黑色食物（黑芝麻、黑豆），温补食物（羊肉、桂圆）
- **情绪**：不恐惧，肾主恐

## 五、节气与儿童发育

| 节气 | 发育关注点 | 书童提醒 |
|:---|:---|:---|
| **立春** | 春季长高黄金期开始 | 保证睡眠、增加户外运动 |
| **惊蛰** | 传染病高发期 | 注意卫生、勤洗手 |
| **清明** | 过敏高发（花粉） | 减少户外活动时间 |
| **立夏** | 紫外线增强 | 防晒、戴帽子 |
| **小满** | 湿气重 | 饮食清淡、少吃生冷 |
| **夏至** | 昼长夜短，睡眠减少 | 强制午休、晚上早睡 |
| **立秋** | 换季易感冒 | 及时增减衣物 |
| **白露** | 早晚温差大 | "春捂秋冻"，不要过早穿厚衣 |
| **秋分** | 秋季长高第二黄金期 | 补充营养、适量运动 |
| **立冬** | 户外活动减少 | 室内运动替代、注意通风 |
| **冬至** | 日照最短，情绪易低落 | 增加光照、补充维生素D |
| **大寒** | 最冷时节 | 保暖防冻、预防感冒 |

## 六、书童的节气陪伴

### 每个节气可以做什么？

**讲一个故事**：
- 节气的由来
- 与节气相关的古诗
- 节气里的农耕故事

**做一个活动**：
- 春分：立蛋游戏（春分昼夜平分，容易立蛋）
- 清明：做青团、踏青
- 冬至：包饺子、数九

**观察自然**：
- 记录当天的温度、天气
- 观察植物的变化（发芽、开花、落叶）
- 观察动物的行为（燕子归来、蝉鸣、大雁南飞）

**调整作息**：
- 根据节气调整睡眠时间
- 根据节气调整饮食
- 根据节气调整衣物
