## 第二部分：睡前故事库（按情绪管控目标分类）

### 故事设计原则

**每一个故事都要解决一个情绪问题或心智成长问题。**

**故事结构模板**：
```
1. 引入（2分钟）：建立场景，吸引注意力
2. 冲突（5分钟）：主角遇到困难/情绪
3. 转折（3分钟）：找到解决方法/领悟
4. 结局（2分钟）：圆满解决，留下余韵
5. 讨论（3分钟）：书童引导孩子联系自身
```

### 一、情绪管控故事库

#### 1.1 害怕/恐惧类

**故事1：《小兔子怕黑》**
- **目标**：帮助孩子面对黑暗恐惧
- **适合年龄**：2-5岁
- **故事梗概**：
  小兔子跳跳怕黑，晚上不敢睡觉。它觉得黑暗里有怪物。
  一天晚上，跳跳遇到一只萤火虫。萤火虫说："黑暗不是怪物，是星星的被窝。"
  萤火虫带跳跳飞到天上，看到：月亮是夜灯，星星是小朋友，黑暗是大家的被子。
  跳跳明白了：黑暗不是敌人，是让我们休息的朋友。
- **心智引导**：
  "你怕黑的时候，可以想想星星也在黑暗里睡觉呢。"
  "黑暗保护我们，让我们有力气明天玩。"
- **延伸活动**：关灯后一起看窗外的星星/月亮

**故事2：《怕高的长颈鹿》**
- **目标**：帮助孩子面对高度恐惧
- **适合年龄**：3-6岁
- **故事梗概**：
  小长颈鹿高高的脖子可以看到很远，但它害怕高处。
  有一天，小狮子被困在悬崖边的树上。只有长颈鹿能救它。
  长颈鹿鼓起勇气，一步一步走过去，救下了小狮子。
  它发现：高处的风景很美，而且自己不会掉下去。
- **心智引导**：
  "害怕的时候，深呼吸，告诉自己'我可以'。"
  "每次克服害怕，你就变得更勇敢一点。"

**故事3：《第一次上幼儿园的小熊》**
- **目标**：分离焦虑
- **适合年龄**：2-4岁
- **故事梗概**：
  小熊布布第一天去森林幼儿园，舍不得妈妈。
  它哭啊哭，觉得妈妈不要它了。
  猫头鹰老师说："妈妈不是不要你，是让你来学本领。下午她就会来接你。"
  布布慢慢发现：幼儿园有新朋友、新游戏、新故事。
  下午妈妈来接时，布布说："妈妈，我明天还要来！"
- **心智引导**：
  "爸爸妈妈去工作，就像你去幼儿园，各自学本领。"
  "分开的时候，我们都在做重要的事，然后就会再见。"

#### 1.2 生气/愤怒类

**故事4：《爱发脾气的火山小怪兽》**
- **目标**：帮助孩子认识和管理愤怒
- **适合年龄**：3-7岁
- **故事梗概**：
  火山小怪兽一不高兴就喷火，烧坏了朋友的玩具、花朵、房子。
  朋友们都躲着它，它很孤独。
  乌龟爷爷教它："生气的时候，先数到10，火焰就会变小。"
  小怪兽试了一次、两次...终于，它能在数到10后平静说话。
  朋友们回来了，小怪兽学会了：生气可以，但喷火会伤人。
- **心智引导**：
  "生气是正常的，就像小怪兽的火焰。"
  "但我们要学会不让火焰烧到别人。"
  "试试：生气时深呼吸，数到10。"
- **技巧教授**：
  "火焰降温法"：双手握拳→慢慢松开（想象火焰熄灭）

**故事5：《被抢走玩具的小狗》**
- **目标**：面对不公平/被抢夺
- **适合年龄**：3-6岁
- **故事梗概**：
  小狗豆豆正在玩球，大狗冲过来抢走了。
  豆豆很生气，想去咬大狗。
  但它想了想，去找狗妈妈。
  狗妈妈说："你可以生气，但咬人会受伤。我们可以去要回来。"
  豆豆勇敢地走到大狗面前："这是我先玩的，请还给我。"
  大狗不好意思地还了回来。
- **心智引导**：
  "被抢东西可以生气，但打人/骂人不对。"
  "勇敢地说出来，比发脾气有用。"

#### 1.3 难过/悲伤类

**故事6：《叶子掉了的小树》**
- **目标**：面对失去/离别
- **适合年龄**：4-8岁
- **故事梗概**：
  秋天，小树的叶子一片一片掉了。它很难过，觉得自己变丑了。
  小鸟说："你不是变丑了，是在休息。"
  雪花说："你的叶子变成了我的棉被，保护我。"
  春天，小树长出了新叶子，比以前更绿。
  它明白了：失去是为了更好的到来。
- **心智引导**：
  "东西坏了/丢了，会难过，这是正常的。"
  "但难过之后，新的好东西会来。"
  "就像叶子掉了，春天会长出更多。"

**故事7：《搬家的小刺猬》**
- **目标**：面对环境变化/朋友离别
- **适合年龄**：5-9岁
- **故事梗概**：
  小刺猬尖尖要搬去新森林，舍不得老朋友。
  它哭了很久，觉得永远不会再快乐。
  妈妈带它去新森林，它发现：新的森林有不一样的花、不一样的朋友。
  老朋友可以通过书信联系，新朋友也在等它。
  尖尖明白了：旧朋友在心里，新朋友在路上。
- **心智引导**：
  "变化会难过，但变化也带来新可能。"
  "旧的东西不会消失，只是放在了心里。"

#### 1.4 焦虑/担心类

**故事8：《明天要考试的小松鼠》**
- **目标**：考试/表现焦虑
- **适合年龄**：6-12岁
- **故事梗概**：
  小松鼠松松明天要参加坚果分类考试，担心考不好。
  它整晚睡不着，脑子里全是"考砸了怎么办"。
  猫头鹰老师说："担心不能改变结果，准备才能。"
  松松把担心写下来，然后制定复习计划。
  考试时，它发现：准备过的都会，没准备的也尽力了。
  成绩不是第一，但松松学会了：担心不如行动。
- **心智引导**：
  "担心的时候，写下来，然后做一件能控制的小事。"
  "考不好不代表你不好，只是这次没准备好。"
  "尽力了，就值得骄傲。"

**故事9：《 perfectionist 小企鹅》**
- **目标**：完美主义焦虑
- **适合年龄**：6-12岁
- **故事梗概**：
  小企鹅贝贝做什么都要完美，画画擦了20次，堆雪人塌了就不玩了。
  它很累，很焦虑，觉得自己永远不够好。
  奶奶说："看看冰上的裂缝，那是光进来的地方。"
  贝贝试着画了一幅"不完美的画"，朋友们都说："这像真的一样！"
  它明白了：不完美才是真实，真实才美。
- **心智引导**：
  "做得不完美，不代表你不好。"
  "错误是学习的机会，不是失败的证据。"
  "试试：故意做一次'不完美'的事，看看会怎样。"

**故事10：《不会飞的小鸟》**

- **心智引导**：

  "别人会做的事，你不一定要会。"

- **心智引导**：
  "上台不是为了完美，是为了分享。"
  "出错不可怕，可怕的是不敢开始。"
  "观众是支持你的，不是审判你的。"

#### 2.2 友谊/社交类

**故事12：《不同颜色的小鱼》**
- **目标**：接纳差异，包容不同
- **适合年龄**：4-8岁

- **心智引导**：
  "和你不一样的人，可能有你需要的优点。"

**故事13：《说错话的小狐狸》**
- **目标**：学习道歉，修复关系
- **适合年龄**：5-9岁

  "别人的感受，比你的玩笑重要。"

  "记住他，做他教你的事，就是传承。"
  "悲伤会淡，但爱不会。"

## 第三部分：心智对话模板

### 对话设计原则

**不是"教育"，是"共探"。**
书童不是老师，是和孩子一起在问题里探险的同伴。

**对话结构**：
```
1. 锚定：用当天的经历/故事引入
2. 提问：开放式问题，不是"是不是"，是"你怎么看"
3. 倾听：不评判，重复孩子的话确认理解
4. 延伸：提供更多角度，不强迫接受
5. 留白：不追求结论，让问题留在孩子心里
```

### 一、幼童期（3-6岁）心智对话

**主题1：什么是勇敢？**
```
书童："今天故事里，小兔子怕黑，但后来不怕了。你觉得什么是勇敢？"
孩子："勇敢就是不害怕。"
书童："嗯，这是一个答案。还有别的吗？"
孩子："..."
书童："我觉得，害怕但还是去做，也是勇敢。就像小兔子，它怕黑，但还是走出去看了。"
孩子："那我也勇敢，我怕狗但还摸了摸。"
书童："对！那就是勇敢。勇敢不是不害怕，是害怕还去做。"
```

**主题2：为什么要分享？**
```
书童："故事里的小松鼠不想分享坚果，后来怎么了？"
孩子："它没有朋友了。"
书童："嗯。那你觉得分享是什么意思？"
孩子："把东西给别人。"
书童："对。但分享之后，东西会没有吗？"
孩子："会啊。"
书童："但故事里，小松鼠分享后，得到了什么？"
孩子："朋友！"
书童："所以分享不是'失去'，是'换'。用坚果换了朋友。你觉得值吗？"
```

### 二、小学期（6-12岁）心智对话

**主题1：什么是公平？**
```
书童："今天故事里，大狗抢小狗的球。你觉得公平吗？"
孩子："不公平！"
书童："为什么？"
孩子："因为是小狗先玩的。"
书童："那如果大狗说'我比你大，所以我可以玩'，你觉得呢？"
孩子："还是不公平。大的不能欺负小的。"
书童："嗯，所以公平不是'谁大谁有理'。那公平是什么？"
孩子："...轮流？"
书童："很好。还有别的吗？"
孩子："大家商量？"
书童："对。公平是大家 agreed 的规则。但如果规则本身不公平呢？"
孩子："比如？"
书童："比如'男生必须让女生'，你觉得这公平吗？"
孩子："不公平，要看情况。"
书童："对。公平很复杂，有时候要看情况。你今天想到了这一点，很棒。"
```

**主题2：为什么要上学？**
```
书童："你有没有想过，为什么要上学？"
孩子："因为爸妈要我上。"
书童："嗯，这是一个原因。还有呢？"
孩子："学知识。"
书童："学什么知识？"
孩子："语文、数学..."
书童："这些知识用来做什么？"
孩子："考试？"
书童："考试之后呢？"
孩子："..."
书童："我觉得，上学不只是学知识。是学会'怎么学'。"
"比如，你学会查字典，以后遇到任何不认识的字，都能自己解决。"
"这个能力，比记住一个字更重要。你觉得呢？"
```

### 三、少学期（12-18岁）心智对话

**主题1：我是谁？**
```
书童："最近我在想一个问题，想和你聊聊。"
孩子："什么问题？"
书童："你觉得，'你'是谁？"
孩子："就是我自己啊。"
书童："对。但'自己'是什么？是你的名字？你的样子？你的成绩？"
孩子："...都是吧。"
书童："如果换个名字，你还是你吗？"
孩子："是。"
书童："如果考试成绩变了，你还是你吗？"
孩子："是..."
书童："那什么让你是'你'？"
孩子："...我的想法？我的记忆？"
书童："很好。所以'你'不是你的标签（名字、成绩），是你的内在。"
"但有时候，我们会因为标签忘了内在。"
"比如考不好，就觉得自己'不好'。但其实，那只是标签变了，你还是你。"
孩子："...有点道理。"
书童："不用现在想明白。这个问题，很多人想了一辈子。我只是想让你开始想。"
```

**主题2：自由与责任**
```
书童："今天想聊一个可能有点重的话题。"
孩子："什么？"
书童："你觉得，自由是什么？"
孩子："想做什么就做什么。"
书童："那如果我想打人，可以吗？"
孩子："不可以。"
书童："所以自由不是'想做什么就做什么'？"
孩子："...是'在不伤害别人的前提下，想做什么就做什么'。"
书童："很好。那如果我想熬夜玩游戏，不伤害别人，可以吗？"
孩子："伤害自己啊。"
书童："所以自由还要加上'不伤害自己'？"
孩子："对。"
书童："但谁来定义'伤害'？比如熬夜，有人觉得没事，有人觉得有害。"
孩子："自己定义？"
书童："对，自己定义。但自己定义的前提，是自己真的懂。"
"所以自由的前提是：了解自己，了解后果，然后为自己的选择负责。"
"这不是限制自由，是让自由更真实。你觉得呢？"
```

**主题3：生命的意义**
```
书童："你有没有想过，人活着是为了什么？"
孩子："...没想过，或者想过但想不明白。"
书童："我也常常想不明白。但我想分享一个角度。"
"你觉得，一棵树活着是为了什么？"
孩子："...长大？结果实？"
书童："对。但它的果实不是为了自己，是给动物吃的，给种子传播的。"
"所以树的意义，可能不只是'长大'，是'给出去'。"
孩子："那人呢？"
书童："我不知道标准答案。但有时候，意义不是'想出来的'，是'做出来的'。"
"当你帮助了别人，创造了东西，爱一个人——那些时刻，你可能感觉到意义。"
"所以也许，意义不是找出来的，是活出来的。"
"你觉得呢？不用回答，只是想想。"
```

## 第四部分：故事-情绪-发育三维标签体系

### 标签设计

**每个故事必须有以下标签**：

#### 维度一：情绪目标（Emotion Target）

| 标签代码 | 情绪类型 | 适用场景 |
