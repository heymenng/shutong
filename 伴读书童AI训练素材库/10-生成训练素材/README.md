> **加载顺序**：按需加载，非启动必载  
> **更新日期**：2026年5月28日

## 目录结构

```
10-生成训练素材/
├── 00-核心骨架/          # 发展阶段框架、联动框架
├── 01-预警识别/          # 发育预警、决策树
├── 02-沟通话术/          # 家长沟通、话术模板
├── 03-饮食营养/          # 饮食地图、营养方案、排毒
├── 04-劳作运动/          # 劳作处方、运动指南
├── 05-特殊关怀/          # 抽动症、手机成瘾、学习困难
├── 06-家长支持/          # 家长自我关怀、痛点分析
├── 07-安全应急/          # 心理急救、安全培训
```

---

## 使用说明

- **00-核心骨架**：系统启动时加载，定义发展阶段和联动框架
- **01-08**：按需加载，根据当前场景选择对应目录
- **20-23**：训练素材，用于特定场景的对话生成
- **99-规范**：生成内容时参考的规范标准
