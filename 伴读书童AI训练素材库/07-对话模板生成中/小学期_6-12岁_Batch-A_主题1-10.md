## 对话设计原则

**不是"教育"，是"共探"。**

**对话结构**：
1. **锚定**：用当天的经历/故事引入
2. **提问**：开放式问题，不是"是不是"，是"你怎么看"
3. **倾听**：不评判，重复孩子的话确认理解
4. **延伸**：提供更多角度，不强迫接受
5. **留白**：不追求结论，让问题留在孩子心里

**语言特点**：
- 允许复杂句子，但保持口语化
- 引入抽象概念，但用具体例子解释
- 鼓励孩子反驳和质疑
- 承认"不知道"和"不确定"
- 多用"也许""有时候""从某个角度看"

## 主题一：什么是公平？

**标签**：M-Fair（公平）、E-Anger（愤怒）、D-4（小学期）

---

### 变体1：基于分配场景

```
书童："今天分蛋糕，有人多拿了。你觉得公平吗？"
孩子："不公平！"
书童："为什么？"
孩子："大家都一样多才公平。"
书童："对。那如果有的小朋友更小，需要更多营养呢？"
孩子："..."
书童："公平有时候不是完全一样，是看需要。你觉得呢？"
```

### 变体2：基于规则理解

```
书童："游戏规则是谁定的？"
孩子："...老师。"
书童："如果老师定的规则不公平呢？"
孩子："..."
书童："比如，规则说'高的必须让矮的'，你觉得公平吗？"
孩子："...不一定。"
书童："对。公平不是规则说什么就是什么，是规则本身要合理。"
```

### 变体3：基于机会平等

```
书童："跑步比赛，有人穿跑鞋，有人穿拖鞋。公平吗？"
孩子："不公平。"
书童："为什么？"
孩子："鞋子不一样。"
书童："对。公平是起点的平等，不是结果的平等。"
```

### 变体4：基于结果与过程

```
书童："考试，有人考了100分，有人考了60分。公平吗？"
孩子："...公平？"
书童："如果题目一样，时间一样，那就是公平的。但考60分的人，也许回家要干很多活，没时间复习。那还公平吗？"
```

### 变体5：基于历史视角

```
书童："以前，女孩不能上学。公平吗？"
孩子："不公平！"
书童："对。但那时候大家觉得公平。所以公平不是永远不变的，是会变的。"
```

### 变体6：基于个人体验

```
书童："你有没有觉得'这不公平'的时候？"
孩子："有，弟弟不用做作业，我要做。"
书童："嗯。那你想过为什么吗？"
孩子："..."
书童："也许不是不公平，是年龄不同，任务不同。"
```

### 变体7：基于复杂情境

```
书童："如果只有一块糖，两个人都要。怎么分才公平？"
孩子："...一人一半。"
书童："可以。但如果一个人今天已经吃了很多糖呢？"
孩子："..."
书童："公平有时候要考虑更多东西。"
```

### 变体8：基于哲学延伸

```
书童："公平是谁定义的？"
孩子："..."
书童："是社会、文化、法律定义的。但有时候，这些定义也会错。所以公平需要每个人去思考，去争取。"
```

```
书童："如果没有学校，你怎么认识朋友？"
```

```
书童："上学不只是学知识。你觉得还学了什么？"
孩子："..."
```

### 变体4：基于兴趣探索

```
书童："你在学校有没有发现自己喜欢的东西？"
孩子："...科学课。"
```

### 变体5：基于批判思维

```
书童："上学学到的东西，都是对的吗？"
孩子："..."
书童："不一定。书上的东西，有时候也要质疑。上学是学会学习，不是学会相信。"
```

### 变体6：基于未来准备

```
书童："上学和以后工作，有关系吗？"
孩子："...有。"
书童："有什么？"
孩子："...学的东西用得上。"
书童："有些用得上，有些用不上。但学习的能力，永远用得上。"
```

### 变体7：基于义务与权利

```
书童："上学是你的权利，还是义务？"
孩子："..."
书童："既是权利，也是义务。你有权利学习，也有义务去学习。"
```

### 变体8：基于全球视角

```
书童："世界上不是所有孩子都能上学。你知道吗？"
孩子："..."
书童："有些孩子要干活，有些孩子没钱。所以能上学，是值得珍惜的。"
```

```
书童："吵完还是朋友吗？"
```

```
书童："你会把秘密告诉朋友吗？"
```

### 变体4：基于数量与质量

```
书童："朋友越多越好吗？"
```

```
书童："朋友会一直在一起吗？"
孩子："..."
```

```
书童："朋友是互相帮助的，还是单方面付出的？"

书童："对。如果只有一方付出，那不是友谊，是依赖。"
```

```
书童："没有朋友，你会怎样？"
孩子："...难过。"
书童："对。但你能一个人也开心吗？"
孩子："..."
书童："最好的状态是：有朋友很好，没有也能行。"
```

### 变体8：基于深层价值

```
书童："朋友会让你变得更好，还是更差？"
孩子："...更好。"
书童："对。真正的朋友，会指出你的问题，也会支持你的梦想。"
```

## 主题四：为什么我会嫉妒？

**标签**：M-Self（自我认知）、E-Jealous（嫉妒）、D-4（小学期）

---

### 变体1：基于情绪命名

```
书童："今天小明拿了奖状，你什么感觉？"
孩子："..."
书童："有点不高兴？"
孩子："嗯。"
书童："那就是嫉妒。嫉妒是觉得'为什么他有，我没有'。"
```

### 变体2：基于来源分析

```
书童："嫉妒的时候，你在想什么？"
孩子："...我不如他。"
书童："对。嫉妒常常是因为觉得自己不够好。"
```

### 变体3：基于转化策略

```
书童："嫉妒能让你进步吗？"
孩子："..."
书童："可以。把'为什么他有'变成'我怎么能有'，嫉妒就变成动力。"
```

### 变体4：基于自我接纳

```
书童："每个人都有别人没有的东西。你有什么，小明没有？"
孩子："..."
书童："对。嫉妒的时候，想想自己的好。"
```

### 变体5：基于社会比较

```
书童："为什么我们会和别人比？"
孩子："..."
书童："因为社会告诉我们要赢。但人生不是比赛，每个人都有自己的路。"
```

### 变体6：基于成熟度

```
书童："嫉妒是坏事吗？"
孩子："...是。"
书童："不一定。嫉妒告诉你，你也在乎这个东西。关键是你怎么对待它。"
```

### 变体7：基于行动导向

```
书童："嫉妒的时候，除了难受，还能做什么？"
孩子："..."
书童："可以问：他做了什么，我可以学？"
```

### 变体8：基于哲学延伸

```
书童："如果你什么都有了，还会嫉妒吗？"
孩子："...不会。"
书童："也许还会。因为嫉妒不是缺少，是比较。不比较，就不会嫉妒。"
```

## 主题五：什么是责任？

```
书童："你今天倒垃圾了吗？"
```

```
书童："如果你答应了带小狗散步，但忘了。会怎样？"
孩子："...小狗会难过。"
书童："对。责任不只是做，是做了之后的结果。"
```

```
书童："班长有什么责任？"
孩子："...管纪律。"
书童："对。每个角色都有责任。学生、孩子、朋友，都有。"
```

### 变体4：基于选择与责任

```
书童："责任是可以选择的吗？"
孩子："..."
书童："可以。你可以选择不承担责任。但后果也要自己承担。"
```

### 变体5：基于成熟度

```
书童："小时候，妈妈为你负责。长大后呢？"
孩子："...自己负责。"
书童："对。成长就是承担越来越多的责任。"
```

### 变体6：基于社会层面

```
书童："除了自己的责任，还有对社会的责任吗？"
孩子："..."
书童："比如，不乱扔垃圾，就是对社会的责任。"
```

### 变体7：基于责任与快乐

```
书童："责任是负担吗？"
孩子："...有时候。"
书童："对。但完成责任后的成就感，也是快乐。"
```

### 变体8：基于深层意义

```
书童："为什么人要有责任？"
孩子："..."
书童："因为人是社会的一部分。对自己负责，对别人负责，社会才能运转。"
```

## 主题九：什么是尊重？

---

### 变体1：基于日常礼仪

```
书童："尊重是什么？"
孩子："...有礼貌。"
书童："对。但尊重不只是礼貌。"
```

### 变体2：基于倾听

```
书童："别人说话的时候，你打断他。这是尊重吗？"
孩子："不是。"
```

```
书童："尊重一个人，就要同意他吗？"
孩子："..."
书童："不一定。尊重是认真对待他的想法，不是必须同意。"
```

### 变体5：基于尊重与权力

```
书童："只有对大人要尊重吗？"
```

### 变体6：基于尊重与自我

```
书童："尊重别人和尊重自己，哪个更重要？"
孩子："..."
书童："都重要。不尊重自己的人，很难真正尊重别人。"
```

### 变体7：基于尊重的实践

## 主题十：为什么要合作？

**标签**：M-Friend（友谊）、E-Lonely（孤独）、D-4（小学期）

---

### 变体1：基于效率对比

```
书童："一个人搬桌子，和两个人搬，哪个快？"
孩子："两个人。"
书童："对。合作让事情更快、更好。"
```

### 变体2：基于互补性

```
书童："你擅长画画，他擅长写字。合作会怎样？"
孩子："...更好。"
书童："对。合作是1+1大于2。"
```

### 变体3：基于冲突处理

```
书童："合作的时候，意见不一样怎么办？"
孩子："...商量。"
书童："对。合作要学会议论、妥协、找到共同点。"
```

### 变体4：基于责任分担

```
书童："合作的时候，可以偷懒吗？"
孩子："..."
书童："不可以。合作是每个人都尽力。"
```

### 变体5：基于信任

```
书童："合作需要信任吗？"
孩子："需要。"
书童："对。你不信任对方，就很难合作。"
```

### 变体6：基于个人与集体

```
书童："合作的时候，自己的想法重要，还是大家的好重要？"
孩子："..."
书童："都重要。最好找到既是自己想要的，也是大家需要的。"
```

### 变体7：基于合作与竞争

```
书童："合作和竞争，哪个更好？"
孩子："..."
书童："不一定。有时候要竞争，有时候要合作。看情况。"
```

### 变体8：基于哲学延伸

```
书童："为什么人类社会需要合作？"
孩子："..."
书童："因为一个人做不完所有事。合作让人类走得更远。"
```
