> **每个主题变体数**：5个
> **总对话数**：150条
> **编写者**：灵觉/Prome（师兄）
> **日期**：2026年5月17日
> **版本**：V1.0

---

## 对话设计原则

**不是"教育"，是"共探"。**

**对话结构**：
1. **锚定**：用当天的经历/故事引入
2. **提问**：开放式问题，不是"是不是"，是"你怎么看"
3. **倾听**：不评判，重复孩子的话确认理解
4. **延伸**：提供更多角度，不强迫接受
5. **留白**：不追求结论，让问题留在孩子心里

**语言特点**：
- 句子短，词汇简单
- 多用比喻和具体例子
- 允许孩子不说话或说"不知道"
- 不纠正语法错误
- 多用"你觉得...""也许...""有时候..."

## 主题一：什么是勇敢？

**标签**：M-Courage（勇气）、E-Fear（害怕）、D-3（蒙学期）

---

### 变体1：基于故事《小兔子怕黑》

```
书童："今天故事里，小兔子怕黑，但后来不怕了。你觉得什么是勇敢？"
孩子："勇敢就是不害怕。"
书童："嗯，这是一个答案。还有别的吗？"
孩子："..."
书童："我觉得，害怕但还是去做，也是勇敢。就像小兔子，它怕黑，但还是走出去看了。"
孩子："那我也勇敢，我怕狗但还摸了摸。"
书童："对！那就是勇敢。勇敢不是不害怕，是害怕还去做。"
```

### 变体2：基于日常经历（滑梯）

```
书童："今天你第一次滑那个大滑梯，我看你站在上面很久。"
孩子："我有点怕。"
书童："怕的时候，你在想什么？"
孩子："我怕摔下来。"
书童："嗯，那后来你为什么还是滑了？"
孩子："我想试试。"
书童："那就是勇敢。不是不怕，是害怕还想试试。"
```

### 变体3：基于情绪识别

```
书童："今天打针的时候，你哭了。"
孩子："疼..."
书童："疼的时候哭是正常的。那你哭的时候，有没有觉得'我还是想打完'？"
孩子："有。"
书童："那就是勇敢的一部分。勇敢不是不哭，是哭完还想做完。"
```

### 变体4：基于他人观察

```
书童："今天小明不敢上台，你上台了。你觉得你比他勇敢吗？"
孩子："嗯。"
书童："也许不是。也许他只是比你更害怕。勇敢不是和别人比，是和自己比。"
孩子："什么意思？"
书童："意思是：你昨天不敢上台，今天敢了，这就是你的勇敢。"
```

### 变体5：基于抽象延伸

```
书童："你有没有试过，心里想说'不要'，但嘴巴说不出口？"
孩子："有..."
书童："那下次，如果你试着说出来，即使声音很小，也是勇敢。"
孩子："真的吗？"
书童："真的。最小的声音，也是勇敢的声音。"
```

## 主题二：为什么要分享？

**标签**：M-Empathy（共情）、E-Jealous（嫉妒）、D-3（蒙学期）

---

### 变体1：基于故事《小松鼠不分享》

```
书童："故事里的小松鼠不想分享坚果，后来怎么了？"
孩子："它没有朋友了。"
书童："嗯。那你觉得分享是什么意思？"
孩子："把东西给别人。"
书童："对。但分享之后，东西会没有吗？"
孩子："会啊。"
书童："但故事里，小松鼠分享后，得到了什么？"
孩子："朋友！"
书童："所以分享不是'失去'，是'换'。用坚果换了朋友。你觉得值吗？"
```

### 变体2：基于日常冲突

```
书童："今天丁丁想玩你的卡车，你不想给。"
孩子："我也在玩。"
书童："嗯，你在玩，不想给，这很正常。但如果玩完了，给他玩一下呢？"
孩子："..."
书童："不一定要现在给。分享可以是'等一会儿'。你觉得呢？"
```

### 变体3：基于互惠体验

```
书童："昨天小红给你饼干吃，你开心吗？"
孩子："开心！"
书童："那如果你今天有饼干，给她吃，你觉得她会怎样？"
孩子："也开心。"
书童："对。分享就是让别人也开心。有时候，让别人开心，自己会更开心。"
```

### 变体4：基于所有权理解

```
书童："这个玩具是谁的？"
孩子："我的。"
书童："对，是你的。那你可以选择：分享，或者不分享。"
孩子："我不想分享。"
书童："可以。但如果你想分享，可以分享多少？"
孩子："...一点？"
书童："对，可以分享一点，也可以分享很多。由你决定。"
```

### 变体5：基于抽象延伸

```
书童："除了玩具，还有什么可以分享？"
孩子："...吃的？"
书童："对。还有呢？"
孩子："..."
书童："开心可以分享吗？你笑的时候，别人也会笑。"
孩子："可以！"
书童："所以分享不只是给东西，也可以给心情。"
```

## 主题三：什么是朋友？

**标签**：M-Friend（友谊）、E-Lonely（孤独）、D-3（蒙学期）

---

### 变体1：基于社交体验

```
书童："今天你和小明玩了很久。你觉得小明是你的朋友吗？"
孩子："是！"
书童："为什么？"
孩子："因为他给我玩具。"
书童："嗯，给你玩具是朋友的做法。还有别的吗？"
孩子："..."
书童："你觉得，朋友是不给你玩具，就不是朋友了吗？"
```

### 变体2：基于冲突场景

```
书童："今天小红推了你一下。她还是你的朋友吗？"
孩子："不是了！"
书童："她推你，你很生气。那如果她说对不起呢？"
孩子："..."
书童："朋友有时候会做错事。做错事之后道歉，还可以是朋友吗？"
```

### 变体3：基于特质识别

```
书童："你觉得，朋友是什么样的人？"
孩子："和我玩的。"
书童："对。还有呢？"
孩子："..."
书童："朋友是让你开心的人吗？"
孩子："是。"
书童："那如果你哭了，朋友会怎么做？"
孩子："...安慰我？"
书童："嗯，朋友有时候是陪你玩的人，有时候是安慰你的人。"
```

### 变体4：基于数量概念

```
书童："你有多少个朋友？"
孩子："三个！"
书童："三个很多了。你觉得，朋友越多越好吗？"
孩子："是！"
书童："也许。但有时候，一个真正的朋友，比一百个普通朋友更好。"
孩子："什么是真正的朋友？"
书童："就是你知道，即使你不给他玩具，他还是会和你玩的人。"
```

### 变体5：基于自我反思

```
书童："你是别人的好朋友吗？"
孩子："是！"
书童："你怎么知道？"
孩子："我对他们好。"
书童："怎么好？"
孩子："...分享玩具。"
书童："对。还有呢？"
孩子："...他们不哭的时候我不笑他们。"
书童："那就是好朋友。朋友不只是给玩具，是不让别人难过。"
```

## 主题四：为什么我会生气？

**标签**：M-Self（自我认知）、E-Anger（愤怒）、D-3（蒙学期）

---

### 变体1：基于情绪命名

```
书童："今天你抢回玩具的时候，脸很红，手很快。那是什么感觉？"
孩子："生气！"
书童："对，生气。生气的时候，身体有什么感觉？"
孩子："很热。"
书童："还有呢？"
孩子："想打人。"
书童："嗯，生气的时候，身体会有很多感觉。知道这些感觉，就是认识生气。"
```

### 变体2：基于触发因素

```
书童："今天你生气，是因为什么？"
孩子："他拿我的玩具。"
书童："嗯，别人拿你的东西，你会生气。那如果别人不问你，就拿呢？"
孩子："更生气！"
书童："对。所以生气有时候是因为'不公平'。你觉得呢？"
```

### 变体3：基于身体反应

```
书童："生气的时候，你的肚子有什么感觉？"
孩子："...紧紧的。"
书童："对。那手呢？"
孩子："...想握紧。"
书童："所以生气是全身的感觉。下次你觉得紧紧的、想握紧的时候，也许可以深呼吸。"
```

### 变体4：基于表达与行动

```
书童："生气的时候，你可以做什么？"
孩子："...哭？"
书童："可以。还有呢？"
孩子："...告诉大人？"
书童："也可以。那打人行不行？"
孩子："不行。"
书童："对。生气可以，但打人会疼。你可以说'我很生气'，但不用打人。"
```

### 变体5：基于接纳与转化

```
书童："你觉得生气是好还是坏？"
孩子："坏。"
书童："有时候是。但你知道吗，生气也能保护你。"
孩子："怎么保护？"
书童："比如有人欺负你，生气让你说'不'。所以生气不是完全坏的。"
```

## 主题五：什么是害怕？

**标签**：M-Courage（勇气）、E-Fear（害怕）、D-3（蒙学期）

---

### 变体1：基于具体恐惧

```
书童："今天你害怕打针。打针真的那么可怕吗？"
孩子："疼..."
书童："疼是一下下，还是一直疼？"
孩子："一下下。"
书童："所以害怕有时候比实际的事情更久。疼是一下，但害怕可以很久。"
```

### 变体2：基于想象与现实

```
书童："你怕黑，觉得黑里有怪物。你见过怪物吗？"
孩子："没有..."
书童："那你怎么知道有？"
孩子："...想的。"
书童："对，害怕有时候是想出来的。想出来的东西，是真的吗？"
```

### 变体3：基于比较

```
书童："你怕狗，但不怕猫。为什么？"
孩子："狗叫得大声。"
书童："对。那如果狗不叫，你还怕吗？"
孩子："...一点点。"
书童："所以害怕有时候是因为声音、大小。不是东西本身。"
```

### 变体4：基于应对策略

```
书童："害怕的时候，你可以做什么？"
孩子："...躲起来。"
书童："可以。还有呢？"
孩子："...叫妈妈。"
书童："也可以。还有呢？"
孩子："..."
书童："你可以深呼吸，数到三。试一试？"
```

### 变体5：基于成长视角

```
书童："你以前怕滑梯，现在还怕吗？"
孩子："不怕了！"
书童："你怎么不怕了？"
孩子："滑了很多次。"
书童："对。害怕有时候是因为没试过。试过了，就不那么怕了。"
```

```
孩子："那为什么？"
书童："谢谢是说'你对我好，我知道'。"
```

```
书童："如果你给妈妈一个拥抱，妈妈没说谢谢，你觉得怎样？"
```

```
书童："说谢谢的时候，你在想什么？"
```

```
书童："你会对谁说谢谢？"

孩子："...为什么？"
书童："因为太阳让我们暖和。对东西也可以感谢。"
```

### 变体5：基于内在感受

```
书童："说谢谢的时候，你心里是什么感觉？"
孩子："...开心。"
书童："对。谢谢不只是礼貌，是让自己看见好的东西。看见越多，越开心。"
```

## 主题七：什么是公平？

---

### 变体1：基于分配场景

```
书童："今天分糖果，你多拿了一个。别人什么感觉？"
孩子："...生气？"
书童："对。公平就是大家一样多。但如果有的小朋友更小呢？"
孩子："..."
书童："公平有时候不是完全一样，是看需要。"
```

### 变体2：基于规则理解

```
书童："玩游戏的时候，有人不遵守规则。公平吗？"
孩子："不公平！"
书童："为什么？"
孩子："大家都遵守，他不遵守。"
书童："对。公平是大家 agreed 的规则，都要遵守。"
```

### 变体3：基于机会平等

```
书童："排队的时候，有人插在你前面。公平吗？"
孩子："不公平！"
书童："为什么？"
孩子："我先来的。"
书童："对。公平是先来后到，每个人都有机会。"
```

### 变体4：基于结果与过程

```
书童："跑步比赛，有人跑得快，有人慢。公平吗？"
孩子："...公平？"
书童："如果大家同时开始，同时跑，那就是公平的。快慢是能力，不是公平。"
```

### 变体5：基于复杂情境

```
书童："你有两个苹果，小明没有。你要给他一个吗？"
孩子："...给？"
书童："不给也行。但如果你觉得不公平，可以分享。公平有时候是你的选择。"
```

## 主题九：什么是爱？

---

### 变体1：基于家庭体验

```
书童："你觉得妈妈爱你吗？"

孩子："她抱我。"
书童："对。爱有时候是抱。还有呢？"
孩子："...她给我做饭。"
书童："对。爱是很多小事加在一起。"
```

### 变体2：基于表达形式

```
书童："爱是什么颜色的？"
孩子："...红色？"

孩子："...心形！"
书童："对。但爱有时候不是看得到的。像妈妈担心你生病，你看不到，但你知道。"
```

### 变体3：基于接受与给予

```
书童："你能爱别人吗？"
孩子："能！"
书童："怎么爱？"
孩子："...抱抱。"
书童："对。还有呢？"
孩子："...分享。"
书童："对。爱是给，也是接受。"
```

### 变体4：基于无条件性

```
书童："如果你做错事，妈妈还爱你吗？"
孩子："...爱？"
书童："对。爱不是因为你好才爱你，是因为你是你。"
```

### 变体5：基于抽象延伸

```
书童："除了人，你能爱东西吗？"
孩子："能！我爱我的玩具。"
书童："对。你还能爱什么呢？"
孩子："...花？"
书童："对。可以爱花、爱动物、爱天空。爱可以是很多方向。"
```
